PLOT3D

nozzle.g,nozzle.q .... ASCII, 2D format files.  
                       To load these with the
                       plot3d loader you must
                       turn off Auto-Detect and
                       set to Multi-Grid, 2D.

nozzleb.g,nozzleb.f .. Binary version of
                       nozzle.g and nozzle.q 
                       These will load using
                       Auto-Detect.

jetflow.g,jetflow.f .. Plot3D Grid and function
                       files (binary).  Auto-detect
                       will work with these.
