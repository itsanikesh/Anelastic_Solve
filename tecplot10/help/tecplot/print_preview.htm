Print Preview

Print Preview allow you to see a preview of your screen image as it will be rendered for the printer is generated.

 

You may:

 

  • Check for problems with intersecting objects which typically do not show up in OpenGL-rendered screen images.

  • Access to the Print Render Options dialog, where you may improve sorting by selecting Force Extra Sorting for All 3D Frames.

 

Note: Vector graphics formats do not support translucency. Print Preview does not display translucency.
