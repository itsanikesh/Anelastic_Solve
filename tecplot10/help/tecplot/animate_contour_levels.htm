Animate Contour Levels

Use the Animate Contour Levels dialog to display all, or a specified subset, of the contour levels in the current data set, one at a time. You specify a start level (the first level you want displayed), an end level, and a level skip to use in the animation sequence.

 

To have Tecplot create a movie file containing the animation, select the Create Animate Movie File check box. To begin the animation, click Animate.

 

The following options are available:

  • Start Level Number: Enter the initial level to display.

  • End Level Number: Enter the last level to display.

  • Level Number Skip: Enter the level skip.

Use one to display all contour levels from Start Level to End Level. Use two to display every other level from Start Level to End Level. Use three to display every third level, and so on.

  • Current Contour Levels: Shows the current contour level displayed. Use the up or down arrows to manually step through the contour levels.

  • Animate: There are three options: On Screen, to AVI file, to RM file. If On Screen is selected the animation will occur on the screen only. If AVI or RM are selected, the appropriate Exporter dialog appears when you click Animate. This is followed by the file dialog to specify the file name and path for the movie file. Finally, the animation will be sent to the file only and not appear on the screen.
