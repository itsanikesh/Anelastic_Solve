Load Data File Options Vars

This page allows loading variables by name or by position.

 

Loading by position is the appropriate option if your variables appear in the same order in each of your data files. Tecplot will retain your variable order.

 

Use Select By Name to select variables by name. An asterisk (*) may be used to match any character. (This is useful for data files with large numbers of variables.)

If you choose to load variables by name, Tecplot variables are created based on the variable names in the data files. If your data files have the same variable names, Tecplot's first variable is created from the first variable common to all data files, then Tecplot's second variable is created from the second variable name common to all data files, and so on. Alternatively, if your variable names vary among your different data files, the Load Variables by Name dialog leads you through the process of matching variables, file by file and variable by variable.

 

To do this:

  1. Select a variable in the Data File Variables list, and click Move>> to move it to the Variables to Load list.

  2. To combine variables with different names from the different files, select a variable in the Data File Variables list, and select the variable in the Variables to Load list that you want to match it to. Click Combine>>.

  3. Asterisks indicate variables that have not been matched in every data file. Any variable that is loaded without matching contents from a file will be filled with zeros for all zones that came from that file.

 

The <<Remove and <<Remove All buttons allow you to delete selected items from the Variables to Load list.
