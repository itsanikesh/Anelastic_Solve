Load Data File Options Zones

On the Zones page, you can specify which zones to load from the data files. The zones are listed in the order in which the data files were chosen (that is, all of the zones from the first file, then all of the zones from the second file, and so on). By default, all of the zones are selected to be loaded.

 

You can click, click-and-drag, Ctrl-click, or Shift-click to highlight the collection of zones you want to load.

 

Use Select By Name to select zones by name. An asterisk (*) may be used to match any character. This is useful for data files with large numbers of zones.

 

If you have selected only specific zones, and want them renumbered when they are read into Tecplot, select the Collapse Zone List check box.
