Translate/Magnify

Translate changes the viewing position of the data or paper. Magnify changes the view magnification of the data within the frame or the paper within the workspace.

 

Choosing this option calls up the Translate/Magnify dialog and changes the cursor mode to Translate/Magnify. While in Translate/Magnify mode, drag the cursor to move the data with respect to the frame, or Shift-drag to move the paper with respect to the workspace. You can rescale your image by pressing "+" to enlarge, "-" to reduce. If you are Shift-dragging to move the paper, the rescale buttons "+" and "-" will enlarge or reduce the paper so long as you have the mouse button depressed. When you release the mouse button, "+" and "-" revert to magnifying the data. The amount of growth or shrinkage for each key press is determined by the Step Size (%) drop-down in the lower part of the dialog.

 

The Translate/Magnify dialog has two regions:

 

Upper Region: Tools for doing data translations of the plot view. The upper region has the following controls:

  • Translation Arrows: These arrows control the direction of the data translation of your image. There are four translation arrows (Up, Down, Right, and Left). To translate the image in a direction by the current step size click on any translation arrow in the specified direction.

  • Note: When translating, your plot is reduced to an approximate wire frame representation.

  • Step Size (%): This drop-down controls the amount of change per translation that occurs each time. To change the step size either enter a new value (between 0.1 and 100) in the Step Size (%) text field, or select one of the following default values from the Step Size (%) drop-down: 5, 10, 20. These default values are set and modified in the Step Size column in the Size Preferences dialog.

 

Lower region: This region contains tools for controlling the view magnification factors of the image. This region has these controls:

  • Magnification Factor: Controls the magnification factor for the view; the default is the current magnification factor. A magnification factor of 1.0 makes the data points just fit within the frame (equivalent to Data Fit). A magnification factor of 0.5 makes the image in the frame one half as large as the Data Fit option. After changing the magnification factor, the object is still in the same general location with respect to the frame, only larger or smaller. Changing the magnification factor preserves the axis scales with respect to one another regardless of the graphics state (X-Y plot, 2- or 3-D) or axis mode (independent, dependent, or XY-dependent).

  • Step Size (%): Controls the amount of change per magnification when you are increasing your magnification. To change the step size either enter a new value (between 0.1 and 100) in the Step Size (%) text field or select one of the following default values from the Step Size (%) drop-down: 1, 10, 20. These default values are set and modified in the Step Size column in the Size Preferences dialog.
