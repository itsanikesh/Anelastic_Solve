Vector Length

Use the Vector Length dialog to control the lengths of vectors. All vectors may be of constant length, or they may have length proportional to the vector magnitude.

 

The following options are available:

  • Relative (Grid Units/Magnitude): Select this to display vectors with length proportional to the vector magnitude. Enter the scale factor in terms of grid units per unit of vector magnitude.

  • Relative (cm/Magnitude): Select this to display vectors with length proportional to the vector magnitude. Enter the scale factor in terms of screen centimeters per unit of vector magnitude.

  • Uniform: Select this to have all vectors one length, regardless of their magnitudes. If you select this option, you can enter a magnitude (as a percentage of frame width) in the text field. The default is two percent.

  • Recalculate Length: Reset vectors to a reasonable default based on the size of the longest vector. The result is a relative vector length with the scale factor expressed in grid units per unit of vector magnitude.
