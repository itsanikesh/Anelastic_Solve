Polar Drawing Options

The Polar Drawing Options dialog allows you to select how Tecplot draws a line between two points for the Polar Line plot type.

 

A line between two points in the Polar Line plot type may be drawn as a straight line, or as an interpolation of Theta-R values. By default, lines are drawn straight.

Select Polar Line Drawing Mode

  • Curved Lines (Theta-R Interpolation): The connection between two points is a curve. This may slow plotting speed for large data sets. (Tecplot stops drawing a line that involves too many revolutions.) May slow plotting speed for large data sets.

  • Straight Lines (X-Y Interpolation): Draw straight lines between points for the Polar Line plot type. (Default.)

  • Angle to Approximate Curved Lines

 

Enter a value to control the approximate used to draw arcs in the Polar Line plot type. These arc include the axes, gridlines, and curved lines drawn in Theta-R Interpolation mode.
