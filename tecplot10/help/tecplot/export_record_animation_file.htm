Export Record Animation File

The following options are available:

  • Record Next Image: Selecting this option will record the current image of the frame(s) you specified on the AVI Exporter or Raster Metafile Exporter dialogs (the current frame, all frames, or the work area) to be added to your animation file. Each step of the animation is stored as a temporary file. The entire animation is assembled and created when you click Finish Animation.

  • Finish Animation: After you have modified your data, this creates the appropriate AVI or Raster Metafile animation file by assembling all of the frames recorded with Record Next Image (above).

 

The Record Animation File dialog is accessed by selecting the AVI or Raster Metafile export formats from the Export option.

 

After you have made your selections from the options on the AVI Exporter or Raster Metafile Exporter dialogs (for instance, image region and size in pixels), click OK, and the Select Export File dialog will appear.

 

When you have specified a file name and location for your file, click Save. The Record Animation File dialog will appear.
