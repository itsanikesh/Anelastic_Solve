extract fe boundaries

Use the Extract FE Boundary dialog to extract the boundary of a finite-element zone.

 

This following options are available:

  • Source Zone: Select the source zone from the drop-down. This zone must be finite-element. To select an FE-volume zone, the active window must be in the 3D Cartesian plot type.

  • Retain boundaries between blanked and unblanked cells: Select this check box to retain the boundaries bordering blanked and non-blanked cells.

 

If the check box is selected, Tecplot creates a boundary zone of the blanked zone as if it were a full zone. That is, the boundary will be the set of all cell faces that form the boundary between blanked regions and non-blanked regions as well as the non-blanked portions of the actual zone boundary.

 

If this check box is not selected, Tecplot creates a boundary based on the original data (without blanking) and then removes parts of the boundary that correspond to the cells that are blanked. The boundary between blanked and non-blanked cells is not extracted. This is useful for extracting a portion of a boundary (such as the interior boundary only).

 

For 3-D volume data (FE tetrahedral or FE brick in 3D Cartesian mode), the boundary of a zone is its outer surface. The Extract FE-Boundary dialog allows you to extract the outer surface of an FE-volume zone. The resulting zone will be an FE-surface zone with Quadrilateral element type.

 

For FE triangle or quadrilateral zones, the extracted boundary is an FE line segment zone.
