Contour Details Labels

All Contour Details dialogs include the following options:

  • C1, C2, C3, C4: Select which contour group is viewed and modified in the Contour Details dialog.

  • Var: Select the contour variable for the selected contour group.

  • More>>: Expand the Contour Details dialog to include additional options.

  • Less<<: Contract the Contour Details dialog to include only the contour group selection and the contour variable selection controls.

Use the Labels page of the Contour Details dialog to control the appearance and placement of contour level labels on contour line plots. This dialog has four main regions – at the top, a section that controls whether labels are shown, whether the labels shown use the contour number or the contour value, and the format of the contour values, if those are used; the second section controls the color, font, size, and background fill for the labels; the third section controls automatic label generation and alignment; the bottom section sets a toggle for alignment of user-positioned labels and provides a button for clearing all labels.

 

The top section has the following options:

  • Show Labels: Select this check box to display contour labels.

  • Note: Flood Type on Contour page of the Zone Style dialog must be set to either Lines or Lines and flood for contour labels to show on the plot.

  • Use Contour Number: Select this to use contour numbered labels if labels are shown. A contour legend may be placed on the plot to display the contour value that corresponds to the contour numbered label.

  • Use Contour Value: Select this to use contour value labels if labels are shown. With contour value labels, the contour lines are labeled with the value of the contour variable along that line. The format of the label is determined by the Format drop-down, and possibly the Precision or Custom Set text fields.

  • Number Format: Choose the format from the Specify Number Format dialog which appears.

 

The font and fill controls are as follows:

  • Color: Choose a color from the Select Color dialog.

  • Font: Choose a font from the drop-down of Tecplot's basic fonts.

  • Size (%): Specify the size of the contour labels as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 1.5, 2, 3, 6, 10. You can alter these default values from the Size Preferences dialog.

  • Fill Behind: Select this check box to color a filled box surrounding the contour label.

  • Fill Color: Choose a color for the filled box, if Fill Behind is selected.

  • Fill Margin: Specify the margin between the contour label and the edge of the fill box. The margin is specified as a fraction of the character height of the contour-value label. You can enter a value in the text field, or use the up and down arrows to increase or decrease the margin.

 

The automatic label controls are as follows:

  • Generate Automatic Label (with each Redraw): Select this check box to have Tecplot automatically generate labels.

  • Align Labels: Select this check box to have Tecplot align automatically generated labels along the contour lines they label.

  • Spacing: Enter the spacing of the contour labels along the contour line, as a percentage of the frame.

  • Level Skip: Enter the number of levels to skip in generating the automatic labels; the default value of one means to label every contour level.

 

The final controls are as follows:

  • Align Next User-Positioned Label: Select this check box to have user-positioned labels aligned along the contour lines they label. This check box does not affect labels already placed, but only those placed after it is selected (or deselected)

  • Clear All Contour Labels: Click to clear all contour labels from the current plot. A confirmation dialog appears, prompting you for confirmation.
