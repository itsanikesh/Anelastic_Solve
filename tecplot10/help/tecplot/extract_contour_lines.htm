Extract Contour Lines

Use the Extract Contour Lines dialog to extract plotted contour lines as zones. Your data will be altered by the creation and naming of new zones.

 

You have the option to:

  • Create a single zone from all independent line segments. With this option a new zone will be created for each contour line plotted. The number of new zones will equal the number of contour levels.

  • Create a separate zone for each independent line segment in each zone. With this option you may create many more zones than there are contour levels. New zones are created in each source zone for each topologically independent contour line.

 

Click Extract to proceed with the extraction and the creation of new zones.

 

Click Cancel to stop the extraction and keep the data unaltered.

 

The created zones are FE-line segment type zones. After generating the zones, it is recommended that you plot them as meshes. Scatter and/or vector plots may also be used.
