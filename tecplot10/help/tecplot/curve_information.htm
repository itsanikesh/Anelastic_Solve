Curve Information

Use the Curve Information dialog to view details of fitted curves and also to extract the curve's coefficients or fitted points to a file.

 

The following options are available:

  • Mapping: Select from the drop-down the map for which you want information, or from which you want to extract coefficients or data points.

  • X-Axis Var: Number and name of the X-axis variable for the chosen map.

  • Y-Axis Var: Number and name of the Y-axis variable for the chosen map.

  • Zone: Number and name of the zone for the chosen map.

  • Curve Details: Detailed list of coefficients used in the equation to draw the line.

  • Write Curve Details to File: Creates an ASCII file with the coefficients for each curve fit or spline.

  • Write Data Points to File: Creates an ASCII file with the data points for each curve fit or spline.

 

To view coefficient and other curve information:

  1. Select Curve Info from the Data menu. The Curve Information dialog appears with the coefficient information.

  2. Read the desired information.
