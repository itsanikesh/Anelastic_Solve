Vector Zone Layer

Select the Vector check box to turn on the Vector zone layer.

 

Vector plots are field plots of the direction and magnitude of vector quantities. The vector quantities can be displacements, velocities, forces, or anything else representable as vectors. Two-dimensional vector plots require that two variables be assigned as the X- and Y-vector components; 3-D vector plots require three variables (the third represents the Z-vector component). The vector field may be used to compute particle trajectories called streamtraces.
