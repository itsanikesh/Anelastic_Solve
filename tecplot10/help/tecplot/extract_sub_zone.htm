extract sub zone

Use the Create SubZone dialog to create a replica of a portion of an existing zone.

 

To create the sub-zone, click Create.

 

The following options are available:

  • Source Zone: Select the zone to duplicate from the drop-down.

  • Index Ranges from Source Zone: For each of I-index and J-index (and, in the 3D Cartesian plot type for K-index), enter appropriate values in the following fields:

  • Start: Specify the start of the range to be duplicated.

  • End: Specify the end of the range to be duplicated. The default value, Mx, means the maximum index.

  • Skip: Specify the range skip, n. Every nth point is duplicated. The default value, one, indicates that every point from Start to End is duplicated. Using a greater skip allows you to decrease the resolution of the duplicated zone.
