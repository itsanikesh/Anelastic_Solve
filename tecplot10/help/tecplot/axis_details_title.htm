Axis Details Title

Use the Title page of the Axis Details dialog to control whether, how, and which titles are displayed for the axis.

 

The following options are available:

  • Show Title On: Select the appropriate check boxes for title display. Options depend upon the plot type setting and include Axis Line, Inner Circle, Outer Circle, All R-Axes, Grid Border Left, Grid Border Right, Grid Border Top, Opposite Edge, Grid Border Start, Grid Border End.

  • Color: Choose the color for the tick mark labels from the drop-down of Tecplot's basic colors.

  • Font: Choose the font and size for the tick mark labels using the Select Font dialog, or use the up/down arrows to increase/decrease the font size.

  • Offset from Line (%): Enter the offset of the tick mark labels from the axis.

  • Position along Line (%): Change where the axis title is positioned along the axis line. For example, in an XY Line plot, if you want the axis title to be on the left side of the X-axis, specify a position near zero.

  • Use Variable Name: Select this to display the name of the assigned variable to the axis as the axis title.

  • Use Text: Select this to specify that the text entered in the unlabeled text field immediately below it should be used as the axis title.
