Zone Style Scatter

Use the Scatter page of the Zone Style dialog to control 2- and 3-D Scatter plots. Each zone is listed, along with its name and its current scatter attributes.

 

To modify a zone's attributes:

  1. Select the zone.

  2. Click on the appropriate attribute's column header. A drop-down appears with the available attribute values.

  3. Choose the appropriate value from the menu.

 

To select a range of zones:

  1. Click on the first (or last) zone.

  2. Shift-click on the last (or first) zone in the range.

 

To select an arbitrary collection of zones:

  1. Click on the first zone.

  2. Ctrl-click on subsequent zones until all desired zones are selected.

 

To edit the group number:

  1. Choose the Edit Group Number option under Zone Grp. An Enter Value dialog appears.

  2. Enter a new group number for the selected zones.

  3. Click OK to modify the group number.

 

To enter a different symbol shape in the Enter ASCII Character dialog:

  1. Enter a character in the text field.

  2. Chose one of the four character fonts: Base, Greek, Math, User-Defined.

  3. Click OK

 

For additional information, please refer to the Tecplot Users Manual.
