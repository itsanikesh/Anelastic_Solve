Extract Discrete Points

This option is used to extract points from a plot data set. To turn on the Extract Points mode, select the Extract Points menu option.

 

To select points:

  1. Click mouse button 1 at each location you would like to extract a point from.

  2. Double-click button 1 on the last data point or press Esc to end.

 

To specify how to save the data, use the Extract Data Points dialog.
