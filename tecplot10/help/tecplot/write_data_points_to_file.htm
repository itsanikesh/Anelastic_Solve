Write Data Points to File

This option creates an ASCII file with the data points for each curve fit or spline.

 

To write data points to file:

  1. Choose Curve Information from the Data menu. The Curve Information dialog appears.

  2. Click Write Data Points to File. The Write Data Points to File dialog appears.

  3. Specify a file name.

  4. Click OK (Save in Windows).

 

The resulting file is a standard Tecplot ASCII data file which can be read into Tecplot to create a zone.
