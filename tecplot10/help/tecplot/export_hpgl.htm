Export HPGL

HP-GL is primarily for pen plotters. Screen colors are panned to pen numbers. You can import these files into some applications..

 

The following options are available:

  • Pen Plotter: Calls up the Pen Plotter Device Configuration dialog. Use it to specify the plotter speed and pen assignments for plotters that accept HP-GL output formats.

  • Force Extra Sorting for 3D Frames: Selecting this check box will cause Tecplot to use extra sorting in all 3-D frames.
