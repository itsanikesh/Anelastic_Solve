Sidebar Layers

Tecplot’s zone and map layers are ways of representing a frame’s data set. A complete plot is a sum of all its active layers, axes, text, geometries, and other elements added to the basic data plotted in the layers.

 

For the 2D and 3D Cartesian plot types, Tecplot’s zone layers are:

  • Mesh.

  • Contour.

  • Vector.

  • Scatter.

  • Shade.

  • Boundary.

 

For the XY Line plot type Tecplot’s map layers are:

  • Lines.

  • Symbols.

  • Bars.

  • Error Bars.

 

For the Polar Line plot type Tecplot’s map layers are:

  • Lines.

  • Symbols.

 

For the Sketch plot type there are no layers.
