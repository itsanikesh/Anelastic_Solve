Contour Details Coloring

All Contour Details dialogs include the following options:

  • C1, C2, C3, C4: Select which contour group is viewed and modified in the Contour Details dialog.

  • Var: Select the contour variable for the selected contour group.

  • More>>: Expand the Contour Details dialog to include additional options.

  • Less<<: Contract the Contour Details dialog to include only the contour group selection and the contour variable selection controls.

Adjustments to the color map for individual frames are made from the Contour Coloring Options dialog. You may set the color distribution method to continuous or banded. You may cut off the color map such that coloring outside a given range is not performed. You may reverse the color map. Your color map may contain multiple cycles.

 

Color Distribution Method

The color distribution method determines how to look up the colors in the global color map.

  • Banded: A solid color is assigned for all values within the band between two levels. For contour flooding this means the area between two levels is filled with a single solid color. For multi-color scatter symbols or vectors this means all scatter symbols or vectors that have a contouring value between two contour levels will be assigned the same color.

  • Continuous: Assigns linearly varying colors to all multi-colored objects or contour flooded regions. Continuous coloring is currently only available for 2D and 3D Cartesian plot types. Selecting Continuous enables the Contour Values at Color Map Endpoints fields, where you may set the minimum and maximum values.

  • Contour Values at Color Map Endpoints: These text fields assign contour values to associate with the start and end of the color map when a continuous color distribution method is in use. By default these are set to the min and max of the contour variable for all active zones.

If you specify a minimum value that is larger than the minimum contour value, the color at the end of the spectrum is used for coloring all objects that have a contour value less than the minimum value entered in the text field. The same principle applies to the maximum value.

 

Color Cutoff

This lets you specify a range within which contour flooding and multi-colored objects, such as scatter symbols, are displayed. Contour flooding outside the range you specify in the text fields will not be plotted.

 

Color Map Adjustments

  • Reverse: Selecting this check box will reverse the color map.

  • Cycles: Enter a number of times to cycle through all the colors of the color map in each region.
