Label Points and Cells

Use the Label Points dialog to control whether and how data points in your plot are labeled. You can label the individual data points (or nodes), or label the cells of finite element data, using either the node or cell index, or the value of a variable at the specified point.

 

The Label Points dialog has one set of options for 2- and 3-D plots, as well as a set for XY and Polar Line plots.
