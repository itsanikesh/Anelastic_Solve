Export Overview

Use the Export dialog to write the current frame or layout into a variety of output formats, including several types of raster files and Encapsulated PostScript.

 

Tecplot supports two main types of export files—vector graphics and raster graphics. In vector formats, the plot is drawn as a series of lines. In raster formats, the plot is drawn pixel-by-pixel at a resolution you specify. Vector output frequently results in smaller files, but only raster output supports continuous color gradients and 3-D translucency.

 

Choose a format from the Export Format drop-down, as follows:

  • Audio Visual Interleaved (AVI): A Windows movie file format containing one or more raster images.

  • Windows Bitmap (BMP). A Windows raster image format.

  • Encapsulated PostScript (EPS): Special type of PostScript file designed for inclusion in other applications. May include a vector or a raster image, plus a raster preview image.

  • Flash: Flash is a movie file format commonly used on the Internet. Unlike AVI, Flash supports twenty-four-bit “true color,” which may give better results for 3-D shaded or translucent plots. Additonal help is available here.

  • HP-GL: Mainly for pen plotters. Screen colors are panned to pen numbers. You can import these files into some applications.

  • HP-GL/2: Mainly for pen plotters, HP LaserJet II or IV, PaintJet XL.

  • PostScript (PS): Suitable for direct printing, but unsuitable for importing into other applications. It is recommended that you use the Encapsulated PostScript (EPS) format for importing into other applications. May include a vector or a raster image.

  • Portable Network Graphic (PNG). A raster format commonly used in the World Wide Web.

  • Joint Photographic Experts Group (JPEG). A raster format commonly used in the World Wide Web. Includes an adjustable amount of compression, which may vary image quality.

  • Raster Metafile: A NASA Raster Metafile. Used for creating movies for Framer.

  • Sun Raster: A Sun Microsystems' Sun Raster image.

  • Tagged Image File Format (TIFF): A raster format commonly used for importing into other applications. You may select Color or Monochrome. For monochrome, you may select an image depth.

  • Windows Metafile (WMF): A vector format used to import into various Windows applications.

  • X-Window System: An image in X-Window Raster (.xwd) format.

 

The dialog displays additional options for the selected format. When the desired options have been selected, click OK. The Select Export File dialog appears. Use this dialog to specify a file name and file path to save the exported data into.
