Data Set Info Sharing

The Data Set Information dialog, accessed from the Data Set Info option on the Data menu, gives summary information about the current data set. You can modify the data set title, zone and variable names of any data set.

 

The Sharing page of the Data Set Information dialog offers the following options:

  • Zone: Use the drop-down to select the desired zone.

  • Variable: Use the drop-down to select the desired variable.

  • Variable is Shared in Zone(s): This list box displays all zones which share the selected variables with the selected zone.

  • Connectivity Shared with Zone(s): This list box displays all zones which share the connectivity list with the selected zone.
