Copy View

Use the Copy View menu option to copy the current frame view to the frame view buffer, where it can then be pasted to other frames having the same plot type.

 

The copied view includes all the attributes of the view that are affected by View menu – the amount of zoom, translation and scale, and, in the 3D Cartesian plot type, the amount of rotation and perspective projection.

 

Transferring Views Between Frames

You can transfer a view between frames by selecting Copy View to copy the view of a plot in one frame (to put the view in the view buffer) and then selecting Paste View to paste the copied view in another frame.

 

For example, suppose there are two frames which use the same data set. One frame has a vector plot and the other frame a contour plot. You zoom into some region in the vector plot and now would like to look at the same view in the contour plot frame. To do this, make the frame with the contour plot the current frame and then select Copy View to copy this view. Next go to the other frame and make it the current frame. Now, select Paste View menu option to paste the contents of the frame view buffer to the new current frame.
