Print Setup

The Print Setup dialog is accessed through the Print dialog and specifies format and print options for Tecplot printing.

 

The following formats are available; each format has its own set of options:

  • PostScript.

  • HP-GL.

  • HP-GL/2.

 

PostScript Print Setup

Use the PostScript Print Setup dialog to choose options for PostScript printing.

 

The following options are available:

  • Color: Select this check box to prepare output using Color PostScript; otherwise, use monochrome output.

  • Spooler Cmd: Enter the command to spool print output in this text field. Use the @ sign to specify wildcard file names. For example, if you print files using the command lpr filename, where filename is the name of the file to be printed, you should enter lpr @ in the Spooler Command text field.

  • Startup String: Enter any necessary startup string in this text field. For example, if your device requires PostScript Level 2, you may need the following setpagedevice command:

  •  %!PS-Adobe-3.0^010<</PageSize [792 1224]>> setpagedevice^010

  • Mopup String: Enter any necessary mopup string in this text field. A mopup string is any string required by your printer to understand that the Tecplot print job is complete and the printer should now be ready to accept additional jobs.

  • Extra Precision: Enter the number of decimal places (0 to 8, inclusive) to which plotting calculations are stored. The default is 0, which should be adequate for most situations.

 

HP-GL Print Setup

Use the HP-GL Print Setup dialog to choose options for HP-GL printing.

 

The following options are available:

  • Spooler Cmd: Enter the command to spool print output in this text field. Use the @ sign to specify wild card file names. For example, if you print files using the command "lpr filename," where file name is the name of the file to be printed, you should enter "lpr @" in the Spooler Command text field.

  • Startup String: Enter any necessary startup string in this text field.

  • Mopup String: Enter any necessary mopup string in this text field. A mopup string is any string required by your printer to understand that the Tecplot print job is complete and the printer should now be ready to accept additional jobs.

  • Pen Plotter: Click this to view or modify the plotter's speed and pen assignments. You can assign specific colors to specific pens, and also assign specific plot objects to specific pens. For example, you can assign axes to one pen, and grid lines to another.

 

HP-GL/2 Print Setup

Use the HP-GL/2 Print Setup dialog to choose options for HP-GL/2 printing.

 

The following options are available:

  • Color: Select this check box to prepare color output; otherwise, use monochrome output.

  • Spooler Cmd: Enter the command to spool print output in this text field. Use the @ sign to specify wild card file names. For example, if you print files using the command "lpr file name," where file name is the name of the file to be printed, you should enter "lpr @" in the Spooler Command text field.

  • Startup String: Enter any necessary startup string in this text field. For example, the following startup string allows the HP LaserJet III printer to print HP-GL:$E%-1B.

  • Mopup String: Enter any necessary mopup string in this text field. A mopup string is any string required by your printer to understand that the Tecplot print job is complete and the printer should now be ready to accept additional jobs.

  • Extra Precision: Enter the number of decimal places (0 to 8, inclusive) to which plotting calculations are stored. The default is 0, which should be adequate for most situations.
