Tools Group Select

Use the Group Select dialog to specify which types of objects within the specified selection region should be selected. All objects having a specified type, or having a set of specified attributes, and lying completely within the selection rectangle (rubber band box) are selected when you click OK.

 

The Group Select dialog offers check boxes to specify the following object types (if the selection rectangle includes none of some object, its associated check box is inactive):

  • Text.

  • Geometries.

  • Frames.

  • Zones.

  • Axis Grid Area.

  • Contour Labels.

  • Streamtraces.

 

The Group Select dialog offers the following attribute filters (again, if the selection rectangle includes none of the appropriate object type, its associated check box is inactive):

  • Geoms of Type: Choose geometries of a particular type. You choose the desired geometry type from the drop-down.

  • Geoms with Line Pattern: Choose all geometries having a particular line pattern. You choose the desired line pattern from the drop-down.

  • Text with Font: Choose all text displayed in a particular font. You choose the desired font from the drop-down.

  • Objects with Color: Choose all objects of a particular color. You choose the appropriate color from the Select Color dialog.
