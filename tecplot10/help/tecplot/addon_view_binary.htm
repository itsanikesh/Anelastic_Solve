Addon View Binary

ViewBin (View Tecplot Binary Data File)

The ViewBin add-on allows you to view the information in a Tecplot binary data (.plt) file.

 

The ViewBin dialog has the following option:

Show Raw Data: Select this option to view zone data.

 

Limitations

On some machines the font used to display the header information may not be a mono-pitched font and consequently some of the results may not line up directly below the table header.
