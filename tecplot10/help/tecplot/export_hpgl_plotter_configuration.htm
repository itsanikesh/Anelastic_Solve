Export HPGL Plotter Configuration

Use the Pen Plotter Device Configuration dialog to specify the plotter speed and pen assignments for plotters that accept HP-GL output formats.

 

The following options are available:

  • Plotter Speed (cm/sec): Set the pen speed of the plotter, using the units centimeters per second.

  • Color/Pen: Use the Color/Pen columns to assign a pen number to each of Tecplot's colors. The default is Pen 1 for each color.

  • Object/Pen: Use the Object/Pen columns to assign a pen number to each of several different Tecplot object types. Using these pen assignments, you can assign one color to axes, another to zone boundaries, and so forth.

 

The following object types can have pen assignments:

  • Axis.

  • Labels.

  • Major Grid.

  • Minor Grid.

  • Multi-Color.

  • Stream.

  • Zone Bounds.
