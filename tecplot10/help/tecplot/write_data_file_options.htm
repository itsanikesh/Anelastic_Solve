Write Data File Options

Use this dialog to specify which portions of the current data set to write, as well as the output format. By default, the output format is binary data in BLOCK format.

 

You may specify one or more zones, one or more variables, and choose which of the following record types are written:

  • Text.

  • Geometries.

  • Custom Labels.

  • Field Data.

  • Data Sharing Linkage (If Possible).

  • Face Neighbor Information Generated by Tecplot.

 

The Write Data File Options dialog has the following controls:

  • Text: Select this check box to write text records.

  • Geometries: Select this check box to write geometry records.

  • Custom Labels: Select this check box to write custom label records.

  • Field Data: Select this check box to write zone records.

  • Data Sharing Linkage (If Possible): Select this check box to write data sharing links.

  • Face Neighbor Information Generated by Tecplot: Select this check box to write all face neighbor information.

 

Save Data File Using

  • ASCII: Select this check box to write the data set as an ASCII human-readable data file.

  • Binary: Select this check box to write the data set as a Tecplot binary data file.

  • Precision: The number of significant digits for each value written to the ASCII data file. (Range one to 18.)

 

Zone/Geometry Format

  • POINT (ASCII only): Select this check box to write your ASCII data in POINT format, all variable values are stored data point by data point. Point format is not allowed if a variable selected is cell-centered or is of Bit type.

  • BLOCK (ASCII only): Select this check box to write your ASCII data in block format. All variable values are stored in block, one block for each variable.

  • Zone(s): Select the zone you want to write from the scrolled list.

  • Variable(s): Select the variables you want to write from the scrolled list. By default, all variables are selected.

 

Layout Association

  • Associate Layout with Newly Saved Data File: Select this check box to reference the internal data set file information with the newly written file.

 

When you have specified a file name either click OK or press Enter. Once you have specified a file name and clicked OK or pressed Enter, the Write Data File dialog appears. From that dialog, you can specify which portion of the data set to write and also specify the output format.
