Tools Streamtrace Termination

Select the Add Streamtrace Termination Line tool to add a streamtrace termination line interactively by drawing a polyline in the current data region. The drawn polyline ends any streamtraces that pass through it.

 

To draw a Streamtrace Termination Line:

  1. Move the cursor into the data region.

  2. Click once at the desired starting point for the line.

  3. Click again at each desired break point.

  4. When the polyline is complete, double-click on the last point of the polyline, or press Esc on your keyboard.
