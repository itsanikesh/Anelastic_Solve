Axis Details Range

Use the Range page of the Axis Details dialog to specify the axis range, position, and dependency options for plotting within a frame.

 

The Range page contains various regions (from top to bottom) below the common Show Axis and Page buttons regions.

 

The items in regions varies slightly depending upon the plot type. The Range pages for the 2D Cartesian, XY Line, and Sketch plot types all have the same items, including the Grid Position (%) fields and an active Axis Position (%) field.

 

The Range page for the 3D Cartesian mode has the Size Factors fields and the Reset Size/Dependency button instead of the Grid Position(%).

 

For further information, please refer to the Tecplot 10 User Manual.
