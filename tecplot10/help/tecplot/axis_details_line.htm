Axis Details Line

Use the Line page of the Axis Details dialog to control whether the axis line is displayed, and if so, in what color and thickness. This page also controls the edge assignment of 3-D axes.

 

The following options are available:

  • Show Axis Line: Select this check box to display the axis.

  • Show Line on Opposite Edge (3D Cartesian only): If selected, a line the same color and thickness as the axis line is drawn on the edge opposite the main axis edge. For auto-edge assignment, this will result in both edges of the grid area hexagon having axis lines. Note: This line is always drawn if any of the faces adjacent to it have grid lines or are filled, or are part of the axis box. In that case, this toggle has no effect.

  • Color: Choose the color for the axis line from the Select Color dialog.

  • Thickness (%): Specifies the thickness of the line pattern as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, 1.5. You can alter these default values from the Size Preferences dialog.

  • Show Axis Box (3D Cartesian only): If selected, all edges of all axes are shown.

  • Auto 3-D Edge Assignment (3D Cartesian only): When selected, the axis line edge is determined by Tecplot for all axes. The determining factor is which of the two axis edges that is part of the grid area hexagon (defined by the three back planes of the axis box) is closer to the bottom of the frame.

  • Show Viewport Border (2D Cartesian, XY Line, Sketch only): If selected, a line is drawn around the viewport.

  • Show Grid Border (Polar Line only): If selected, a line is drawn around the grid area (the lines R=RMin, R=RMax, Theta=ThetaMin, and Theta=ThetaMax). This line is drawn using the line color and thickness of the axis line.

  • Align Axis with: Specifies where to draw the Theta-axis.

  • X-Axis Value: Defines the R-value for aligning the Theta-axis. The axis is draw from Theta=Theta-Min to Theta=Theta-Max along the circle defined by R-value. If the R-value is less than R-Min, R-Min circle is used. If the R-value is more than R-Max, R-Max is used.

  • Offset (%): Enter the offset of the line from the axis.

  • Position (%): Change where the axis line is positioned.
