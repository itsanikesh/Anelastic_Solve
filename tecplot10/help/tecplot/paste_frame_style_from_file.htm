Paste Frame Style from File

This option allows you to utilize a previously saved Tecplot stylesheet.

 

Selecting this option calls up the Paste Style from File dialog where you can specify which previously store style sheet you want to use in this instance.

 

To restore a previously stored Tecplot stylesheet:

  1. Specify a stylesheet name. In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. In Motif, the URL may be entered into the Selection text field.

  2. Click OK (Save in Windows).

 

To specify which portions of the saved style are restored, click Options.
