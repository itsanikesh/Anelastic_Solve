Triangulate

Use the Triangulate dialog to create a finite-element surface zone from one or more arbitrary source zones. The source data zones should all lie in the same plane. Triangulation preserves the accuracy of the data: the resulting zone has the source data points as nodes and consists of a set of Triangle elements.

 

The following options are available:

  • Source Zone(s): Select the zone or zones to triangulate. If you select more than one zone, they are combined into one new triangulated zone with any coincident points ignored. The zones can be ordered or finite-element, although typically they will be irregular data in I-ordered zones. To select a single zone, click on it in the scrolled list Select Zone(s).

- To select a range of zones:

1. Click on the first or last zone in the range.

2. Shift-click on the last or first zone in the range.

- To select a set of arbitrary zones:

1. Click on the first zone you want.

2. Ctrl-click on subsequent zones until you have selected all the zones in the set.

  • Use Boundary Zone(s): Select this check box if you want to specify a boundary zone, that is, a zone that defines the boundary of the region to triangulate. This boundary zone must be I-ordered. It can be either a line that surrounds the data, or a line that defines an interior boundary. If you do not select a boundary zone, the triangulation routine assumes that the data is contained within a convex polygon and that all points in the interior can be connected.

  • Include Boundary Points: Select this check box if you want to include boundary points in the source data. If this box is selected, points that are part of the boundary zone are considered part of the source zones, and hence show up in the resultant triangulation.

  • Triangle Keep Factor: Enter a value defining bad triangles. At the completion of triangulation, Tecplot attempts to remove bad triangles from the outside of the triangulation. This does not include any triangles next to the boundary zone, only those along the edges where there is no boundary (or the boundary zone points are excluded). The definition of a bad triangle is stored as the Triangle Keep Factor as a number between zero (three collinear points) and 0.5 (an equilateral triangle). Values between 0.1 and 0.3 are typical.

 

To perform the triangulation, click Compute. Tecplot displays a dialog while the triangulation is in progress, informing you of its progress.

 

To abort the triangulation, click Cancel on this dialog.

 

When the triangulation is complete, the data set includes a new FE-surface zone (with Triangle element type) titled "Triangulation." The number of triangles in the new zone is reported along with the number of coincident points ignored.
