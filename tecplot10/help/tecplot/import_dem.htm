The DEM Data Loader

 

DEM Data Loader can load Digital Elevation Map files that have the same file format as the U.S. Geological Survey's standard DEM format. These files are generally used by cartographers, geologists, and geographers to map terrain or geography. The DEM Data Loader will not accept Spatial Data Transfer Standard (SDTS) formatted data. DEM files are available on the Web for a number of states in the United States.

 

For more information, please refer to the following references:

  • General: http://edc.usgs.gov/geodata/

  • User's Guide: http://edc.usgs.gov/products/elevation/dem.html

 

After choosing one or more DEM files to load, you are presented with a simple dialog where you can set the I- and J-skipping. Since DEM files are quite large you will likely want to set both of these to be ten or more.
