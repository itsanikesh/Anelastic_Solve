Load Complex Data File Warning

If you select Load Data File(s) when the current data set is used by any frames besides the current frame, the Load Data File Warning dialog appears with four options:

  • Create New Data Set and Reset Current Frame Style: Select this to create a new data set in the current frame with the style sheet attributes redefined to new frame defaults.

  • Create New Data Set and Retain Current Frame Style: Select this to create a new data set and keep the style sheet attributes in the current frame.

  • Add to Current Data Set: Select this to attach the new specified data file into the current data, in the current frame.

  • Replace Current Data Set with New Data Set: Select this to substitute the new data set everywhere the current data set is used while retaining the current data set's style sheet attributes.
