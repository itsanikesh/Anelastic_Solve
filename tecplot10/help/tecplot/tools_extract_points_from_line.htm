Tools Extract Points from Line

Select the Extract Line tool to extract points along a specified polyline to a data file or a new zone.

 

To select points:

  1. Click your left-hand mouse button at each location where you would like to extract a point.

  2. To end extraction, either double-click on the last point, or press Esc.

  3. The Extract Data Points dialog appears; use it to specify how many points to extract and how to save the data.
