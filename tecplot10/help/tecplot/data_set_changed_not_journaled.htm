This dialog allows you to save a layout file.

 

Click Change File Name to call up the Write Data File dialog.

 

To change the assigned file name given to this data set:

  1. Click Change File Name. The Write Data File dialog appears.

  2. Select the drives and directories to save this file in, using the Directories file drop-down (Motif), or Save in drop-down (Windows).

  3. Enter the file name for the data set file in the File Name text field.

  4. Click OK (Save in Windows) to save the data set with the new name and dismiss the Enter a Data Set File Name dialog.

 

To save the data set using the assigned file name, click OK without making any changes.

 

The Enter A Data Set File Name dialog appears if you attempt to save a layout without previously saving the current data set using Write Data File. Use this dialog to specify the file name under which Tecplot will save and recognize an internally created data set (such as a data set created from scratch using 1D, Rectangular, or Circular). This dialog is also used to specify a new name for a data file that has been altered using Specify Equations, Smooth, and so forth, or one that has been added to, for example, by extracting data. The current file name that Tecplot has assigned this unsaved data set is displayed next to the heading Current File.
