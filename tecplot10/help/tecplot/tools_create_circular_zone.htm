Tools Create Circular Zone

Select the Create Circular Zone tool to add new circular zones to the current Tecplot data set.

 

To create a circular zone:

  • Click once in the current data region to specify the center of the zone.

  • Drag until the zone has the desired radius. The new zone created is IJ-ordered.

To specify the maximum I-index and J-index, use the Create Circular Zone dialog that appears.
