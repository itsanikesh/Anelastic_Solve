Open Layout

To open a layout file, either enter a layout file name in the File Name text field or select a file from the Look in drop-down. You open a layout package file using the Open Layout option from the File menu. Note that you may open both standard layout files and layout package files from this dialog. If you have an unsaved layout, Tecplot asks you if you want to save it before opening a new layout file.

 

To show a different set of file names in the Files list box either choose a different directory (using the Look in drop-down in Windows, or the Directories list box in Motif) or specify a different file filter by entering a filter with wildcard symbols in the List Files of Type and File Name text fields.

 

To combine the layout file with the current layout in Tecplot, click on the Append check box prior to clicking OK (Open in Windows). If the Append Layout check box is not selected, the opened layout replaces the existing layout.

 

To open a file using a URL in Windows, you must select the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://.

 

For example, ftp://ftp.microsoft.com/myplot.plt. In Motif, you may open a file with a URL by entering the full URL path in the Selection text field.

 

Ctrl-O is the keyboard shortcut for the Open Layout dialog.
