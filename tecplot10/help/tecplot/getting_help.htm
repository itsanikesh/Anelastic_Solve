Getting Help

You can obtain Help for Tecplot in any of the following ways:

  • Click on Contents on the Tecplot Help menu. The Tecplot Help system appears in your Web browser.

  • Click Help on any Tecplot dialog. The Tecplot Help system appears with information for that dialog.

  • Move the cursor over any menu item, sidebar button, or Quick Edit dialog button. The status line at the bottom of the Tecplot application window gives a brief description of the control.

  • Move the cursor over the Tecplot workspace. The status line gives brief instructions on using the currently selected mode.

  • Move the cursor over any menu item, sidebar button, or Quick Edit dialog button, and press F1. The Tecplot Help system appears with help on the item.

  • Choose Keyboard Shortcuts from the Help menu for a quick listing of Tecplot's keyboard shortcuts.
