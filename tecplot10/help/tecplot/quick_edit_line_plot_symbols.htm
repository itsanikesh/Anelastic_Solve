Quick Edit Line Plot Symbols

This area controls whether symbols are plotted at each data point, and whether those symbols are filled or plain.

 

The following options are available:

  • Symb Show: Display symbol plots for the selected maps. This option allows you to turn off just the Symbols map layer for specific maps, without completely deactivating the maps. The following options are available:

    • Yes: Show the symbol plots for the selected maps. If the Symbols map layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate the Symbols layer.

    • No: Turn off the symbol plots for the selected maps.

  • Fill: Select filled or plain symbols, as follows:

    • Plain: Use un-filled symbols for the scatter plot.

    • Filled: Use filled symbols for the scatter plot.
