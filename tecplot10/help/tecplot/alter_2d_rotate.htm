Alter 2D Rotate

Use the 2D Rotate dialog to rotate 2-D field data about a user specified XY-origin. Unlike 3-D rotation, which does not alter the data but merely the user's view of the data, 2-D rotation actually modifies the data. If you save your data set after performing 2-D rotation, you will have a data set different from your original data.

 

The following options are available:

  •  Angle (deg): Specify the angle of rotation. You can either enter a value, in degrees, in the text field, or select a value from the drop-down. All angles are measured in degrees counter-clockwise from horizontal.

  • X-Origin: Enter the X-coordinate of the origin of rotation, that is, the point around which the data is rotated. The default is zero.

  • Y-Origin: Enter the Y coordinate of the origin of rotation, that is, the point around which the data is rotated. The default is zero.

  • Select Zone(s) to Rotate: Select zones for rotation. The default is no zones selected.

 

To select a single zone, click on it. To select a range of zones:

  1. Click on the first or last zone in the range.

  2. Shift-click on the last or first zone in the range.

 

To select an arbitrary set of zones:

  1. Click on the first desired zone.

  2. Ctrl-click on subsequent zones until all desired zones are selected.

  3. To perform the transformation, click Compute.
