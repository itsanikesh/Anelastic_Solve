Contour Details Legend

All Contour Details dialogs include the following options:

  • C1, C2, C3, C4: Select which contour group is viewed and modified in the Contour Details dialog.

  • Var: Select the contour variable for the selected contour group.

  • More>>: Expand the Contour Details dialog to include additional options.

  • Less<<: Contract the Contour Details dialog to include only the contour group selection and the contour variable selection controls.

Use the Legend page of the Contour Details dialog to control the appearance of a legend for your contour plot. The Legend does not appear unless the contour layer or multicoloring is used.

 

General Controls

  • Show Contour Legend: Select this check box to display a legend on the current plot.

  • Show Header: Select this check box to include the name of the contour variable (and "Level" if continue labels on or plottype=line) as a legend heading.

  • Separate Color Bands: Select this check box to separate the color bands in the legend with black lines. Use this option to visually separate similar colors. If this box is not selected, similar adjacent colors may tend to blur together.

  • Alignment: Select this to display the legend vertically, with a vertical band of colors and/or list of levels and a vertical list of numbers side by side.

  • Anchor: Calls up the Anchor Alignment dialog, where you can specify the anchor point, or fixed point, of the object.

  • Size/Location Controls

  • X (%): Enter the X-location of the legend as a percentage of the frame width.

  • Y (%): Enter the Y-location of the legend as a percentage of the frame width.

  • Resize Automatically: If there are a large number of levels the legend can get very large. When Resize Automatically is selected, it will automatically skip some levels to create a reasonably sized legend.

  • Label At: When the color distribution method is Continuous, you have three options for the labeling increment on the legend—contour levels, specified increment, or color map divisions.

  • Increment: The legend labeling increment when the color distribution method is Continuous, and Label At is Specified Increment, and Resize Automatically is off.

  • Include Cutoff Levels: Select this to show the full contour rage in the legend when Color Cutoff is active.

  • Level Skip: Enter the number of levels between numbers on the legend. This also affects the number of levels between contour labels on the plot. Skipping levels on the contour legend compresses the color bar (if one appears); it does not change the spacing between text entries on the legend. Use Line Spacing for that.

  • Line Spacing: Enter the spacing between contour legend numbers. This does not change the number of entries in the legend, so a large value here creates a large legend. Use Level Skip to reduce the number of entries in the legend.

 

Text Controls

  • Header Font: The Select Font dialog appears where you may select a font for the contour legend text and numbers from the drop-down of Tecplot's basic fonts.

  • Number Font: The Select Font dialog appears where you may select a font for the contour legend text and numbers from the drop-down of Tecplot's basic fonts.

  • Text Color: Select a color for the contour legend text and numbers from the Select Color dialog.

 

Number Format Controls

Calls up the Specify Number Format dialog, where you can set various attributes.

 

Legend Box Controls

  • No Box: Select this to have no box drawn around the legend.

  • Filled: Select this to have a filled box drawn around the legend. Specify the fill color using the Fill Color drop-down.

  • Plain: Select this to have a plain box drawn around the legend.

  • Line Thickness (%): Specify the line thickness of the box as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, 1.5. You can alter these default values from the Size Preferences dialog.

  • Box Color: Select a color for the legend box from the Select Color dialog.

  • Fill Color: Select a fill color for the legend box from the Select Color dialog.

  • Margin: Specify the margin as a percentage of the legend box width.
