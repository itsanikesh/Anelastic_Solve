Curve Fit Spline

Use the Spline Settings dialog to enable Tecplot to fit a spline (a special type of smooth curve) through every data point.

 

This curve type assumes that the points of the dependent variable represent functional values of the independent variable. That is, a spline is drawn through the data points after sorting in increasing values along the independent variable axis. The spline is a single-valued function of the independent variable.

 

To specify whether the spline is clamped or unclamped, use the Curve Settings option on the Curves page of the Mapping Style dialog.
