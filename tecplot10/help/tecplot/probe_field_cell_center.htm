Probe Field Cell Center

The Cell Center page of the Probe dialog, like the Var Values page, lists every variable in the current data set, but lists values at the center of the cell that was clicked on. By default, each variable is shown on a single line, which allows display of about the first ten characters of the variable name and seven significant digits of the variable value.

 

To display longer variable names or see more digits of the value, de-select the check box labeled One Line per Variable. If there are more variables than will fit in one window, use the Scroll Up and Scroll Down buttons.

 

The Cell Center page also displays the zone name and number. An asterisk (*) indicates an interpolated value. If in this field, then the variable location is nodal.

Underneath the Scroll buttons, a text box shows what is probed (the variable and object) and its location. Objects may be a streamtrace, slice or iso-surface.
