Zone Style Shade

Use the Shade page of the Zone Style dialog to control 2- and 3-D Shade plots.

 

Each zone is listed, along with its name and its current shade attributes. To modify a zone's attributes, select the zone, then click on the appropriate attribute's column header. A drop-down appears with the available attribute values.

 

Choose the appropriate value from the menu.

 

To select a range of zones:

1. Click on the first (or last) zone.

2. Shift-click on the last (or first) zone in the range.

 

To select an arbitrary collection of zones:

1. Click on the first zone.

2. Ctrl-click on subsequent zones until all desired zones are selected.

 

To edit the group number:

1. Choose the Edit Group Number option under Zone Grp. An Enter Value dialog appears.

2. Enter a new group number for the selected zones.

3. Click OK to modify the group number.

 

For additional information please refer to the Tecplot Users Manual.
