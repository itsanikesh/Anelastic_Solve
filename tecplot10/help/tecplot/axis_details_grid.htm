Axis Details Grid

Use the Grid page of the Axis Details dialog to specify whether gridlines and minor grid lines are displayed, and the characteristics of such lines when they are displayed.

 

To alter these default values:

  1. Choose Preferences menu from the File menu.

  2. Choose the Sizes option from the Preferences menu.

  3. Go to the Pattern Length column in the dialog. Change the values as you need.

 

The following options are available:

  • Show: Select this check box to show the grid lines.

  • Line Pattern: Select the desired pattern from among the following: Solid, Dashed, DashDot, Dotted, LongDash, DashDotDot.

  • Pattern Length (%): Specifies the length of the line pattern as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.5, 0.8, 2, 3, and 5. You can alter these defaults from the Size Preferences dialog.

  • Thickness (%): Specifies the thickness of the line pattern as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1. 0.4, 0.8, 1.5. You can alter these defaults from the Size Preferences dialog.

  • Gridline Color: Select a color for your gridlines from the Select Color dialog which appears.

  • Gridline Cut-Off (R%) (Polar Line only): The point along the R-axis in which you want to stop drawing Theta-gridlines. In a Polar Line plot the abundance of gridlines at the center may obscure data, in which case you may want to adjust this setting.

  • Gridline Draw Order (All plot types except 3D Cartesian): Drawn gridlines first, before any of the other plotting layers, or last, so they overlay any plotting layers.

  • Precise Dot Grid: By selecting the Show check box in the Precise Dot Grid area, you can generate a grid of dots that form an extension of the tick marks you have specified on the Ticks page.
