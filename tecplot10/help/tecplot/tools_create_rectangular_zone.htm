Tools Create Rectangular Zone

Select the Create Rectangular Zone tool to add new rectangular zones to the current Tecplot data set.

 

To create a rectangular zone:

  1. Click once in the current data region to anchor one corner of the zone.

  2. Drag the diagonal corner until the zone is the desired size and shape. The new zone created is IJ-ordered.

 

To specify the maximum I-index and J-index, use the Create Rectangular Zone dialog that appears.
