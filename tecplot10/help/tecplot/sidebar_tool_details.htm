Sidebar Tool Details

Tool Details calls up whichever dialog most closely reflects the current state of the cursor.

 

If the current tool is either the Selector or Adjustor, the button is labeled Object Details if any object is currently selected; otherwise, it is labeled Plot Attributes. If an object is selected, the dialog controlling that object's attributes is displayed when you click the Object Details button. If any other tool is selected, the button says Tool Details.

 

For example, if you select a legend and then click Object Details, the appropriate legend dialog appears. If no object is selected, and the Selector or Adjustor tools are active, one of the Plot Attributes dialog appears, with the appropriate dialog determined as follows: if the dialog has previously been opened to a particular page, the dialog opens at that page, unless, in the meantime, you have added one or more layers to the plot, in which case the dialog opens to the page for the layer most recently added.
