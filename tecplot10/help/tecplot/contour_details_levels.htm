Contour Details Levels

All Contour Details dialogs include the following options:

  • C1, C2, C3, C4: Select which contour group is viewed and modified in the Contour Details dialog.

  • Var: Select the contour variable for the selected contour group.

  • More>>: Expand the Contour Details dialog to include additional options.

  • Less<<: Contract the Contour Details dialog to include only the contour group selection and the contour variable selection controls.

  • Use the Levels page of the Contour Details dialog to add, remove, or recalculate contour levels. The following options are available:

  • Reset Levels: Click to specify approximate number of levels. Tecplot selects the most appropriate (best) number. An Enter Value dialog prompts you for the number of levels; the current number is the default. Enter a new number or click OK.

  • New Levels: Click to call up the Enter Contour Level Range dialog, which you use to specify a new range and either a number of levels or the delta between levels, or to specify an exponential distribution.

  • Add Levels: Click to call up the Enter Contour Level Range dialog, which you use to specify an additional range of contour levels, and either a number of levels or the delta between levels. You can specify that the added levels have an exponential distribution.

  • Levels: a scrolled list shows all current contour levels. To select a single level, click on it.

 

To select a range of levels:

  1. Click on the first (last) level in the range.

  2. Shift-click on the last (first) level.

 

To select an arbitrary set of levels:

  1. Click on the first level.

  2. Ctrl-click on subsequent levels until all desired levels are selected.

    • Remove Selected Levels: Click to remove selected contour levels.

    • Level to Add: Enter a value for a new contour level in the text field provided.

    • Add Level: Click to add the level specified in Level to Add to the list of Contour levels.

    • Contour Variable (Information Only): Displays the minimum and maximum values of the current contour variable.
