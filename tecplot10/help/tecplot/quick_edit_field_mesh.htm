Quick Edit Field Mesh

This area controls whether the mesh is displayed for selected zones, and if so, using what plot type.

 

The following options are available:

  • Mesh Show Options: Display a mesh for the selected zones. This option allows you to turn off just the mesh layer for specific zones, without completely deactivating the zones. The following options are available:

    • Yes: Show the mesh for the selected zones. If the Mesh layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate the mesh.

    • No: Turn off the mesh for the selected zones.

  • Mesh Type: Choose the mesh type for the selected zones. The following options are available:

    • Wire Frame: Mesh lines are drawn underlying all other field layers (Contour, Vector, Scatter, Shade); hidden lines are not removed.

    • Overlay: Mesh lines are drawn above all other field plot layers except vectors and scatter symbols.

    • Hidden: Like Overlay, except that in the 3D Cartesian plot type hidden lines are removed from behind the mesh. In essence, the cells of the mesh are opaque. Surfaces and lines that are hidden behind another surface are removed from the plot.
