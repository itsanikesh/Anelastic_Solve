Tools Slicing

Use the Slicing tool to control your slice rendering interactively.

 

The following options are available:

  • +: If no slices are active this turns on the starting slice; if the starting slice is active the ending slice is turned on.

  • -: If the ending slice is active this turns off the ending slice; if the ending slice is not active this turns off the starting slice.

  • Click: Places the starting slice.

  • Shift-Click: Places the ending slice.

  • Drag: Moves the starting slice.

  • Shift-Drag: Moves the ending slice.

  • 1, 2, 3, 4, 5, 6, 7, 8, 9: Sets the number of intermediate slices.

  • I: Switches to slicing constant I-planes (ordered zones only).

  • J: Switches to slicing constant J-planes (ordered zones only).

  • K: Switches to slicing constant K-planes (ordered zones only).

  • X: Switches to slicing constant X-planes.

  • Y: Switches to slicing constant Y-planes.

  • Z: Switches to slicing constant Z-planes.
