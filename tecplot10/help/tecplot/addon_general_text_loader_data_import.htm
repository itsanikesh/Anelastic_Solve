Addon General Text Loader Data Import

To import data with General Text Loader, follow the instructions below.

 

Start Identification

  • First all-numeric line: If selected, then the data begins at the first line of the file which contains only numbers. Note that if you have specified multiple zones, the all non-numeric lines will also be skipped at the beginning of each zone.

  • First line after line with keyword: If selected, then the data begins at the first non-blank line after the line containing the specified keyword. The keyword is case insensitive.

  • Start at line number: If selected, then the data begins at the specified line number. Blank lines are ignored in the data section.

 

End Identification

  • All lines up to first non-numeric line: If selected, then the data ends at the first non-blank line which contains any text.

  • Stop at line number: If selected, then the data ends at the line the indicated line number. That is, this line number is the last line of data.

  • All lines up to line with keyword: If selected, then the data ends at the first line before the line with the specified keyword. The keyword is case insensitive.

  • End of file: If selected, then the data ends at the end of file.

 

Data Identification

  • Point format: If selected, data is in point format. In this format all values of all variables are given for the first point, then the second point, etc.

  • Block format: If selected, data is in block format. In this format all values for the first variable are given, then all values for the second variable, etc.

  • Data value delimiter: The data value delimiter indicates when the end of a data value has been reached. You can set it to one of the following:

Auto: Space, tab, comma, semicolon.

Tab.

Comma.

Semicolon.

  • Fixed: Each 'width=n' number characters on the line is a token field. White space is removed from the beginning and end of the field. For example, if the line length is 60 and the width is ten, the columns 1-10,11-20,21-30, and so on, are token fields.

  • Width: If the delimiter is 'fixed', enter the width of each field here.

 

Data Dimension

  • Auto-Calculate IMAX: If selected, then the I-dimension is calculated based on the number of data points found. J- and K-max are set to one.

  • Specify Dimensions: If checked, then you can specify the I-, J-, and K-dimensions for the data. There must be enough data points found in the file to match the indicated dimensions.

  • Allow Multiple Zones: If checked, General Text Loader will attempt to read more than one zone from the data file.

  • Zone ends on line with keyword: If 'Allow multiple zones' is checked and 'Auto-calculate' is selected, then you must enter a keyword here to mark the end of one zone and the beginning of the next. Zones are ended when a line containing this text is found.

 

If IMAX is being auto-calculated, then you must specify a keyword which marks the end of one zone and the beginning of the next zone in the file.

 

If the data dimensions are entered, General Text Loader adds zones as necessary depending on the number of data points found in the file. There must be enough data points for each zone. For example, if there are two variables and the IJK-dimensions are (3, 4, 5) and Allow Multiple Zones is checked, there must be an even multiple of (2*3*4*5) data points. That is, there must be n * (2*3*4*5) data points in the file, where n >= 1, and is the number of zones.

 

Data Filters

Ignore non-numeric tokens: If checked, then any non-numeric information in the data sections is ignored. If not checked, General Text Loader displays an error if any non-numeric data is found in the data section.
