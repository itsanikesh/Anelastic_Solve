Copy Frame Style to File

This option uses the Copy Style to File dialog to save the current frame style as a Tecplot stylesheet. Stylesheets typically have the extension .sty.

 

To save the current frame style, while in the Copy Style to File dialog:

  1. Specify a stylesheet name. In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. In Motif, the URL may be entered into the Selection text field.

  2. Click OK (Save in Windows).

 

You can specify which style options are saved. These options are available in the Copy Style Options dialog, which you open by clicking Options.
