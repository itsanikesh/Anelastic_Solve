Scatter Size/Font

Use the Scatter Size/Fonts dialog to control the base font used for ASCII character symbols and the scatter-size variable that can be used to scale scatter symbols.

 

The following options are available:

  • Base Font for ASCII Symbols: Select a font from the drop-down. The following fonts are available:

  • Courier.

  • Courier Bold.

  • Helvetica.

  • Helvetica Bold (Default).

  • Times.

  • Times Bold.

  • Times Italic.

  • Times Italic Bold.

  • Scatter-Size Variable: Select a variable from the drop-down of the data set's variables. If the Scat Size field is set to Size by Variable on the Scatter page of the Plot Attributes dialog, this variable is used to calculate the scatter symbol size at each data point. The actual size of each symbol is determined by multiplying the value of the variable at each point by the Size Multiplier. If the Scat Size field on the Plot Attributes dialog is not set to Size by Variable, this field has no effect.

  • Size Multiplier: Enter the scale factor that multiplies the values of the Scatter-Size Variable to size the scatter symbols. If the Scat Size field on the Plot Attributes dialog is not set to Size by Variable, this field has no effect. Tecplot initially calculates a size multiplier for you, but you can change it at any time. The Size Multiplier multiplied by the scatter variable value gives the size of the scatter symbol at a point, in units specified by the following option buttons:

  • Grid Units/Magnitude: Select this to express the Size Multiplier in terms of grid units per unit of variable magnitude.

  • Cm/Magnitude: Select this to express the Size Multiplier in terms of screen centimeters per unit of variable magnitude.

  • Recalculate Size: Click to reset the Size Multiplier to Tecplot's initial value.
