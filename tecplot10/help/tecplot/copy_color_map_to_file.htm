Copy Color Map to File

Calls up the Copy Color Map to File dialog, in which you specify a file path file name for the color map you want saved.

 

To save your customized color map to a file:

  1. Click OK to close the Color Map dialog after you have customized your color map.

  2. Go to the Workspace menu.

  3. Click on Copy Color Map to File. This calls up the Copy Color Map to File dialog.

  4. Enter in a color map file name (color map files have a .map extension) in the Selection text field File Name list box.

  5. Click OK to write the color map file and close the Copy Color Map to File dialog.
