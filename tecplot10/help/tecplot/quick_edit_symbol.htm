Quick Edit Symbol

This area controls the type and size of symbol displayed in scatter and symbol plots.

 

The following options are available:

  • Symbol Shape: Select the shape of the scatter symbol. The following options are available:

    • Square.

    • Delta.

    • Gradient.

    • Right Triangle.

    • Left Triangle.

    • Diamond.

    • Circle.

    • Other: The Other option calls up the Enter ASCII Character dialog, which includes a text field for specifying the ASCII character to use as the symbol shape, and four option buttons for choosing the character's font: Base, Greek, Math, or User-Defined.

  • Size (Scatter Symbol Size): Specify the size of the scatter symbols as a percentage of the frame width (in the 2D Cartesian plot type) or of the median-axis length (in the 3D Cartesian plot type). You can choose Enter and enter an exact percentage in the Enter Value dialog that appears, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, and 1.5. You can alter these default values from the Size Preferences dialog.
