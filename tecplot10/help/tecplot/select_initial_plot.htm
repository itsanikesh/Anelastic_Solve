Select Initial Plot

This dialog allows you to choose the first plot type to be used for the data set in the current frame.

 

Available options include:

  • Initial Plot Type: Select from Tecplot’s five plot types: 3D Cartesian, 2D Cartesian, XY Line, Polar Line, Sketch, or Automatic. When Automatic is chosen, Tecplot attempts to match the data to the best plot type:

    • If any finite-element volumes or IJK-zones are present, plot type is set to 3D Cartesian.

    • If finite-element surfaces or IJ-zones are present, 2D Cartesian is used.

    • All others are set to XY Line.

  • Show First Zone Only: Displays only the first zone of a plot.

  • Use These Settings for All Data Sets: Applies your selections to any additional plots loaded.
