ijk blanking

IJK-blanking is available only for 3-D volume zones. IJ-blanking is available for 2-D surface zones and uses the same dialog as IJK-blanking. IJK-blanking removes a selected portion of one IJK-ordered zone from the plot. This allows you to create cut-away plots, plots showing the exterior of some data set with a section cut-away to show the interior.

 

You define the blank region by specifying the following:

  • The IJ- or IJK-ordered zone in which the blanking is to be performed.

  • I-, J-, and K-index ranges for the blank region, either using specific index values or percentages of the index range. Only I- and J-index ranges are needed for IJ-blanking.

  • Whether Tecplot should blank the interior or exterior of the defined region.

 

To use IJK-blanking, you must have an IJ- or IJK-ordered zone, and the current frame must be in the 2D or 3D Cartesian plot type. Unlike value-blanking, which operates on all zones within a single frame, IJK-blanking can only be used on a single zone within a frame, and the zone must be IJ- or IJK-ordered.

 

The following options are available:

  • Include IJK-Blanking: Select this check box to turn on IJK-blanking.

  • Domain: Specify the blanking domain by choosing the appropriate option:

  • Interior: Cells within the specified index ranges are blanked; those outside are plotted. This creates a hole in the plotted zone. Use this option to create cut-away plots.

  • Exterior: Cells outside the specified index ranges are blanked; those inside are plotted. This is a convenient way to create a plot of a 3-D sub-zone, without having to create the sub-zone.

  • Zone: Select a zone for the IJK-blanking from the drop-down.

  • Select IJK-Range Using Index Value: Set the format for specifying the IJK-maximum and minimum range values to the index values for the data set range. Tecplot always stores IJK-ranges as percentages, if you use index values to specify the blanking region, Tecplot internally converts to percentages.

  • Select IJK-Range Using % of Max: Set the format for specifying the IJK-maximum and minimum range values as a percentage of the maximum index values for the data set range.

  • I-Start: Start of the I-range.

  • I-End: End of the I-range.

  • J-Start: Start of the J-range.

  • J-End: End of the J-range.

  • K-Start: Start of the K-range. (This option is inactive for IJ-ordered zones.)

  • K-End: End of the K-range. (This option is inactive for IJ-ordered zones.)

 

To use IJK-blanking:

  1. From the Plot menu, choose IJK-Blanking. The IJK-Blanking dialog appears.

  2. Select the Include IJK-Blanking check box to turn on IJK-blanking for the current frame. IJK-blanking does not take effect until this option is turned on, nor are any of the other controls sensitive.

  3. Specify the domain of the IJK-blanking by choosing one of the following option buttons: Interior: Cells within the specified index ranges are blanked. Those outside are plotted. This creates a hole in the zone. Exterior: Cells outside the specified index ranges are blanked. Those inside are plotted. This plots a sub-zone of the zone.

  4. Select the zone to which IJK-blanking is applied from the Zone drop-down. The zone must be IJK-ordered. You may select only one zone at a time.

  5. Specify the format in which you will specify the index ranges by selecting one of the following options: Select IJK-Ranges Using Index Values: If you select this option, you specify the I-, J-, and K-index ranges using actual minimum and maximum indices. Select IJK-Ranges Using % of Max: If you select this option, you specify the I-, J-, and K-index ranges as start and end percentages of the maximum index. For example, you could blank the middle third of a data set by setting the start percentage to 33.3 and the end percentage to 66.6.

  6. Enter the I-, J- and K-index ranges in the fields provided.

  7. When you save a layout, macro, or stylesheet, the IJK-blanking index ranges are stored as the percentage of the maximum index regardless of how you chose to enter them. This way, one file can be used for different zone sizes.
