Iso Surface Details Style

Use the Iso-Surface Details dialog to control Tecplot’s rendering of iso-surfaces. Iso-surfaces are based on a contour variable.

 

The following options are available:

  • Show Mesh: Select this check box to include meshes.

  • Color: Use the Select Color dialog. Multi will result in each mesh having a line color of the contour level corresponding to that mesh.

  • Line Thickness (%): Enter a value in the text field (Range .0001 to 100), or select from five preset choices for mesh line thickness.

  • Show Contour: Select this check box to include contour flooding.

  • Contour Type: Select from Lines, Flood, Lines & Flood, Average Cell Flood, Primary Value Flood.

  • Flood By: Use this drop-down to select the contour group or RGB.

  • Contour Lines By: use this drop-down to select contour groups.

  • Line Color: Select a color for shading from the Select Color dialog.

  • Line Thickness: Enter a value in the text field (between .0001 and 100), or choose from among the five pre-set choices for mesh line thickness.

  • Show Shade: Select this check box to include shading.

  • Color: Use the Select Color dialog to set shading.

  • Use Lighting Effect: Use this drop-down to set shading to Paneled or Gouraud.

  • Use Surface Translucency: Control the amount of translucency used in shaded or contour flooded objects. (A value of one is solid and 99 is nearly complete translucency.)
