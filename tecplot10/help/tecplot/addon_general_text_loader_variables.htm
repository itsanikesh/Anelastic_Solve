Addon General Text Loader Variables

This page deals with options for importing and loading variables.

 

Importing Variables

  • Scan for variable names: If checked, then the loader will scan the file for variable names using the start/end criteria specified.

  • Start line: Enter the starting line of variable names in the file.

  • End line: Enter the ending line of the variable names in the file. This is typically the same as the starting line.

  • Delimiter: The delimiter indicates when the end of each variable name has been reached. You can set it to one of the following:

Auto: Space, tab, comma, semicolon.

Tab.

Comma.

Semicolon.

  • Fixed: Each 'width=n' number characters on the line is a variable. White space is removed from the beginning and end of the field. For example, if the line length is 60 and the width is ten, the columns 1-10,11-20,21-30, and so forth, are variable names. Spaces are removed from the beginning and end of the variable names.

  • Width: If the delimiter is fixed, enter the width of each field here.

  • Variables: If Scan is not checked, then you can enter a list of variable names here. Variable names should be separated by carriage returns.

  • Select Variables to Load: Launches the Loading Variables dialog (discussed below) which allows you to select which variables to load.

 

Loading Variables

  • Variables to Skip: Displays a list of variables that will be skipped.

  • Variables to Load: Displays a list of variables that will be loaded.

  • Move >>: Moves the selected variables to the Variables to Load list.

  • Move All >>: Moves all of the variables to the Variables to Load list.

  • Move <<: Moves the selected variables to the Variables to Skip list.

  • Move All <<: Moves all of the variables to the Variables to Skip list.
