Load Multiple Data Files

Use the Load Data File(s) dialog to select multiple data files for reading into Tecplot. Because you can read multiple files at once, using the Load Data File(s) dialog is a two-step process: first, select a file, then add it to the Selected File(s) list.

 

To select a file and add it to the Selected Files list:

  1. Enter the file name in the File Name text field, or select the file name in the list box below the Look in Label (Windows). In Windows, you may also load multiple files by using Ctrl-click to select additional files. In Motif, enter a file name in the File to be Added text field or select the file name in the Files scrolled list.

  2. Click Add to read in all the desired files in the Selected File(s) list.

  3. In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. For example ftp://ftp.microsoft.com/myplot.plt. In Motif, you may open a file with a URL by entering the full URL path in the Selection text field.

  4. Click OK.

 

To remove a file already in the Selected File(s) list:

  1. Select the file in the Selected File(s) list box.

  2. Click Remove.

 

To read only selected variables, zones, or record types of the selected file:

  1. Select the Specify Options check box.

  2. Click OK.

  3. Use the Load Data File Options dialog to specify which portions of the data set to read.

  4. Select the portions of the file you want read into the frame.

  5. Click OK.
