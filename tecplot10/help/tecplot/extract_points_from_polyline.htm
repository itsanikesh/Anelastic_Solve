Extract Points from Polyline

This option is used to extract points along a specified polyline to a data file or a new zone. To turn on the Extract Line mode, select the Extract Points from Polyline option.

 

To draw the polyline:

  1. Click mouse button 1 at the beginning of the line, and at all desired break  points.

  2. Double-click on the last data point, or press Esc to end.

 

To specify how many points to extract, use the Extract Data Points dialog.

 

To specify how to save the data, use the Extract To options in the Extract Data Points dialog.
