Enter Contour Level Range

Use the Enter Contour Level Range dialog to either create a new set of contour levels, or add a range of contour levels to an existing set. Which action you do is determined by the push-button used to call up the dialog from the Contour Levels dialog: New Range or Add Range.

 

The range can be specified in one of three ways, as follows:

  • Min, Max, and Number of Levels: Select this to specify the minimum and maximum contour levels and the number of contour levels including the minimum and maximum levels. With this option, the contour levels are evenly spaced from minimum to maximum. The current minimum, maximum, and number of levels are displayed; you can accept these values by clicking OK, or enter new values in the appropriate text fields. You may also have Tecplot recalculate the default values for these parameters, by clicking Reset Range.

  • Min, Max, and Delta: Select this to specify the minimum and maximum contour levels and the delta, or change in value between adjacent levels. The current minimum, maximum, and delta are displayed; you can accept these values by clicking OK, or enter new values in the appropriate text fields.

  • Exponential Distribution: Select this to specify the minimum and maximum contour levels and the number of contour levels including the minimum and maximum levels. The minimum and maximum must be either both positive or both negative. With this option, the contour levels are exponentially distributed between the minimum and maximum. The current minimum, maximum, and number of levels are displayed; you can accept these values by clicking OK, or enter new values in the appropriate text fields.

 

Depending on the option button selected, the following text fields are enterable:

  • Minimum Level: Enter the minimum contour level. If Exponential Distribution is selected, it must also have the same sign as the Maximum Value. If you are adding a set of contour levels to an existing set, this is the minimum of the added levels.

  • Maximum Level: Enter the maximum contour level. If Exponential Distribution is selected, it must also have the same sign as the Minimum Value. If you are adding a set of contour levels to an existing set, this is the maximum of the added levels.

  • Number of Levels (Min, Max, and Number of Levels or Exponential Distribution only): Enter the number of contour levels decided. If you are creating a new set of contour levels, this is the total number of levels; if you are adding a set of contour levels to an existing set, this is the number of levels to add.

  • Delta (Min, Max, and Delta only): Enter the change in contour variable value between adjacent contour levels. If you choose a delta larger than the difference between the minimum and maximum, you get only the minimum contour level.

 

The following button is active only if the check box Min, Max, and New Number of Levels is selected:

  • Reset Range: Select this button to have Tecplot recalculate its default choices for the minimum and maximum contour levels, using the default choice for number of levels.
