Quick Edit Field Contour

This area controls whether contours are displayed for selected zones, and if so, using what plot type.

 

The following options are available:

  • Cont Show: Display contours for the selected zones. This option allows you to turn off just the contour layer for specific zones, without completely deactivating the zones. The following options are available:

    • Yes: Show the contours for the selected zones. If the Contour zone layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate.

    • No: Turn off the contour for the selected zones.

  • Cont Type: Choose the contour type for the selected zones. The following options are available:

    • Lines: Plots contour lines (iso-valued lines). If you choose this plot type, you can use the Cont Color attribute to specify Multi-Color to make the line color vary with the contour-variable value. For 3-D volume zones, the Lines option plots wire frames of the iso-surfaces. You can see through such surfaces to the surfaces lying behind them. It is difficult to distinguish a large number of contour iso-surfaces. For the best effect, use Multi-Color for the Cont Color attribute.

    • Flood: Flood the area between adjacent contour lines with a color according to the value of the contour variable, number of contour levels, and the Color Map. With a large number of contour levels, the color flooding appears nearly continuous. For 3-D volume zones, this option plots a contour color flooded plane for each iso-surface in each cell of the zone. This results in a solid colored iso-surface (without mesh or light source shading). A large number of these may be difficult to discern. Three-dimensional sorting is performed on a cell-by-cell basis. Therefore, the iso-surfaces may not sort correctly if you have multiple contour levels within the same cell. To make sure the iso-surfaces sort correctly under these conditions, extract the iso-surfaces as zones (using the Extract Iso-Surfaces option from the Data menu). Then, plot these new zones using plot type Flood. The Flood Trans attribute controls the pseudo-translucency of this plot type.

    • Both Lines and Flood: Contour lines are drawn with color flooding between them. That is, this plot type overlays contour lines on top of contour flooding. For 3-D volume zones, the flooding is done on each iso-surface, and a mesh of the iso-surface is laid on top of the flooding. This results in an overlaid mesh look to the solid colored iso-surfaces. Three-dimensional sorting and light source shading are the same as for Flood plots. The Flood Trans attribute controls the pseudo-translucency of this plot type.

    • Average Cell: Each cell or element is flooded with one solid color based upon the average value of the contour variable at the data points of the cell or element. For 3-D volume zones, this plot type floods each face of each cell with one solid color based on the average of the contour variable at each node of the face. Plotting is faster if an IJK-mode of Face is used for IJK-ordered zones, or if the outer boundary of FE-volume zones is extracted, since each face is flooded, obscuring those cells in the interior. The Flood Trans attribute affects the pseudo-translucency of this plot type.

    • Primary Cell: Each cell or element is flooded with one solid color based upon the primary cell value. For cell-centered data, this is the cell-center value. Otherwise, the value used depends on the zone type and whether the plot is 2-D or 3-D. For 2-D ordered data, the value used is of the (logically) lower-left node of the cell, that is, the cell with minimum I-, J- and K-indices. If the mesh is curvilinear, the lower left corner point may not physically be the lower left corner of the cell. For 2-D finite-element data, the value of the first node in the cell is used. For 3-D volume zones, each face is flooded with a solid color based on the contour value at the minimum I-, J- and K-indices (or the first node of a finite element) for each face of each cell. Plotting is faster, if an IJK-mode of Face is used for IJK-ordered zones, or if the outer boundary of FE-volume zones is extracted, since each face is flooded, obscuring those cells in the interior. The Flood Trans attribute affects the pseudo-translucency of this plot type.
