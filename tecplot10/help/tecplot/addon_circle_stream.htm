Addon Circle Stream

The Circle Stream add-on is used to place a "rake" of streamtraces in a circular pattern.

 

To place a rake of streamtraces in a circular pattern:

  1. Create one or more circle geometries and place them where you want the streamtraces to start.

  2. To set the number of streamtraces that are placed around the circle, edit the Number of Sides field in the Geometry dialog (double-click on the circle to call up the Geometry dialog).

  3. Select the circle geometries you want streamtraces to emanate from.

  4. Launch the Circle Stream tool from the Tools menu.

  5. Choose the direction you want the streamtraces to travel

  6. Click Add Streamtraces.
