Axis Details Ticks

Use the Ticks page of the Axis Details dialog to control the drawing of both axis and minor tick marks.

 

The following options are available:

  • Show Ticks On: Select the appropriate check boxes for ticks display. Options depend upon the plot type setting and include Axis Line, Inner Circle, Outer Circle, All R-Axes, Grid Border Left, Grid Border Right, Grid Border Top, Opposite Edge, Grid Border Start, Grid Border End.

  • Major/Minor Ticks: Select these check boxes to display the tick marks and minor tick marks along the axis.

  • Length (%): Specifies (independently) the length of major and minor tick marks as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.5, 1.2, 2, 3, and 5. You can alter these defaults from the Size Preferences dialog.

  • Thickness (%): Specifies (independently) the thickness of the major and minor tick marks as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1. 0.4, 0.8, 1.5. You can alter these defaults from the Size Preferences dialog.

  • Number of Minor Ticks: Specifies the number of minor ticks between tick marks. Enter an integer value in this text field. The default is four.

  • Direction: Specifies how the tick marks extend from the axis. There are three options:

  • In: Select this option to have the tick marks extend inward from the axis to the interior of the plot.

  • Out: Select this option to have the tick marks extend outward from the axis, away from the plot.

  • Center: Select this option to center the tick marks on the axis.

  • Tick Mark and Label Spacing: Controls the spacing of the tick marks and labels. Options include:

    • Auto Spacing: Select this check box to turn on automatic tick mark and tick mark label spacing. Tecplot automatically determines the number of tick marks and their spacing.

    • Manual Spacing: If Auto Spacing is not selected, two manual spacing options are activated. These options are:

      • Spacing: Enter the distance between tick marks (in grid units) in the field

      • Anchor: Specify an anchor value in the field. The anchor value specifies where one tick mark will be placed, and other tick marks are placed relative to this point.

 

Both the Spacing and Anchor fields are ignored if Auto Spacing is selected.
