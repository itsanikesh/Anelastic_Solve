Export File Name

Use the Select Export File dialog to specify a file name and file path for saving the exported file in the specified export format..

 

To select a file to export, either enter a export file name in the File Name list box, or select a file from the File Name list box. You can choose a different set of files to display in the File Name list box by choosing a different directory (using the Directories drop-down) or specifying a different file filter (using the List Files of Type drop-down).

 

In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. For example ftp://ftp.microsoft.com/myplot.plt.

 

In Motif, save a file with a URL by entering the full URL path in the Selection text field.
