Copy Plot to Clipboard

In Windows, copy the current Tecplot image to the Windows Clipboard.

Note: This feature is not available in Motif.

 

The following options are available:

• Region: Select Current Frame, All Frames, or Work Area from the drop-down. Current frame is the default and captures only the current frame. All Frames captures the smallest rectangular area containing all frames. Work Area captures the workspace.

 

Advanced Options

  • Format: Selecting Vector will create print output using the drawing commands of the printer. The print renders the plot, yielding higher resolution, but some plot options, such as translucency, are not available. Selecting Image will create print output using an image. Rendering is done by Tecplot at the specified resolution, usually less than the printer’s resolution. However, all plot options are available.

  • Force Extra Sorting for 3D Frames: Selecting this check box will cause Tecplot to use extra sorting in all 3-D frames.

  • Color: Select this check box to save the image as a color EPS file.

  • Convert to 256 Colors: Selecting this check box will convert the true color screen image to eight-bit color.

  • Use Width of Image on Screen: Select this option to generate an image file the same size as the current plot on the screen. This option is forced on if you use on-screen image rendering in the Display Performance dialog.

  • Enter Width (pixels): Enter a value in the text field for your exported image’s width. The image region is rendered to the image file to exactly fit a size of Width by Height. (Default is 512.)

  • Antialiasing: Select this option to remove jaggies from the image.

  • Supersample Factor: Control the amount of antialiasing used in the image.
