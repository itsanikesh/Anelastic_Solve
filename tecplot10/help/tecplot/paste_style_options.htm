Paste Style Options

Use the Paste Style Options dialog to specify which portions of a previously saved stylesheet are restored upon reading, and to specify whether the new style will replace the existing style, or be merged with it.

 

Merging replaces the specified portions of the style with the saved settings, but leaves other portions alone. Replacing replaces the specified portions with the saved settings from the file, and resets all other portions to their factory settings.

 

The following options are available:

  • Frame Size and Position: Replace current frame size and position with the values saved in the stylesheet.

  • Plot Style: Replace current plot style with the values saved in the stylesheet.

  • Text: Replace current text with the values saved in the stylesheet.

  • Geometries: Replace current geometries with the geometries saved in the stylesheet.

  • Stream Positions: Replace current streamtrace positions with the streamtrace position values saved in the stylesheet.

  • Contour Levels: Replace current contour levels with the values saved in the stylesheet.

  • Merge with Current Frame Style: Merge saved stylesheet with the current frame style. In this mode, compatible elements (such as text and geometries) are preserved, with the saved elements added. Specified details to load are added, but other details of the current style are left alone.

  • Replace Current Frame Style: Reset current style to defaults, then load the specified portions of the

  • saved stylesheet.
