Export Windows Metafile

Windows Metafile (WMF), is used to import into various Windows applications.

 

The following options are available:

  • Color: Select this check box to save the image as a color WMF file.

  • Region: Select Current Frame or All Frames from the drop-down.

  • Current frame is the default and captures only the current frame.

  • All Frames captures the smallest rectangular area containing all frames.

  • Work Area captures the workspace.

  • Force Extra Sorting for 3D Frames: Selecting this check box will cause Tecplot to use extra sorting in all 3-D frames.
