animate ijk planes

Use the Animate IJK-Planes dialog to display all or a specified subset of the IJK-planes in the current data set, one at a time. You also specify which set of planes to animate, a start index (for the first plane you want displayed), an end index, and an index skip to use in the animation sequence.

 

The following options are available:

  • Planes to Animate: Select the option corresponding to the desired set of planes:

  • I-Planes: Select to animate a set of I-planes. An I-plane is a plane of constant I-index.

  • J-Planes: Select to animate a set of J-planes. A J-plane is a plane of constant J-index.

  • K-Planes: Select to animate a set of K-planes. A K-plane is a plane of constant K-index.

  • Start Index: Enter the index of the first plane to display.

  • End Index: Enter the index of the last plane to display.

  • Index Skip: Enter the index skip. Use one to display all planes from Start Index to End Index. Use two to display every other plane. Use three to display every third level, and so on.

  • Current Index: Shows the current index displayed. Use the up or down arrows to manually step through the index.

  • Animate: There are three options: On Screen, to AVI file, to RM file. If On Screen is selected the animation will occur on the screen only. If AVI or RM are selected, the appropriate Exporter dialog appears when you click Animate. This is followed by the file dialog to specify the file name and path for the movie file. Finally, the animation will be sent to the file only and not appear on the screen.
