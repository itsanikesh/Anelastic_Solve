Axis Details Area

Use the Area page of the Axis Edit dialog to control the formatting of the grid area. There are three regions; one determines the grid area fill, the next describes the attributes of the grid area border, and the last one sets two miscellaneous options.

 

The following options are available:

  • Fill Behind Axes (3D Cartesian only): Select this check box to color fill the grid area.

  • Fill Grid Area (All plot types except 3D Cartesian): When selected, the axis area defined by MinTheta, MaxTheta, MinR and MaxR is filled with a color after filling viewport but before drawing axes or data.

  • Fill Viewport (Polar Line only): When selected, the viewport is filled with a color before drawing grid area, axes, or data.

  • Use Light Source Shading (3D Cartesian only): Use the light source color to color-fill the grid area.

  • 3D Axis Box Padding (3D Cartesian only): Specifies the amount of space between a 3-D object and the axis box as a fraction of the size of the 3-D object.

  • Color: Choose the color for the grid area fill from the Select Color dialog. This option is used only if Fill Behind is selected and Use Light Source is not selected.

  • Viewport Position (%) (All plot types except 3D Cartesian): Sets the viewport position. The viewport clips both the data and the axes.
