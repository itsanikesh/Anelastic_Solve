Create Zone from Values

Use the Create Zone from Values dialog to enter ordered pairs of numbers (X, Y) directly into an I-ordered zone. Use this feature to create simple data sets without leaving Tecplot. You might, for example, want to add a reference curve to some existing plot. You can use this dialog to enter the points defining that reference curve as a zone.

 

The following options are available:

  • Enter Values per Line: Enter up to NV values per line, where NV is the number of variables in the data set. Separate the values with one or more tabs or spaces. All lines must have the same number of values specified and the total number specified must be a multiple of NV.

  • Destination Data Type: Select from the drop-down the appropriate data type for the Tecplot variables created. The default is Auto, which automatically chooses the appropriate data type from the entered data. You can also specify any one of the following types:

o Single.

o Double.

o Long Int.

o Short Int.

o Byte.

o Bit.

 

To create the zone, click Create.
