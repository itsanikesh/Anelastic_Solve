Save Animation

Use the Save Animation dialog to save the current animation as a Raster Metafile or audio visual interleaved (AVI). Raster Metafiles have the extension .rm. Audio-visual interleaved have the extension .avi.

 

Enter a file name in the Selection text field, or select a file from the Files list box (Motif). In Windows, enter a file name in the File name text field, or select a file from the Save in list box. If you select a file name from the Files list box, the new animation writes over the existing file.

 

You can modify the contents of the Files list box by choosing a different directory (using the Directories list box) or specifying a different file filter (using the Filter text field).

 

You can use Framer to view your .rm movie files. In Windows, you may view your .avi movie files by double clicking on the appropriate .avi movie file icon.
