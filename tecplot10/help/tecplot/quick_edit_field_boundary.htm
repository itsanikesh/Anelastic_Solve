Quick Edit Field Boundary

This area controls whether the zone boundary is displayed for selected zones, and if so, what boundary type. Boundary options affect only ordered data sets. Finite-element data has no zone boundaries.

 

The following options are available:

  • Boundary Show: Display boundaries for the selected zones. This option allows you to deactivate some zone boundaries, without completely deactivating the zone. The following options are available:

    • Yes: Show boundaries for the selected zones. If the Boundary zone layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate the Boundary layer.

    • No: Turn off boundaries for the selected zones.

  • Boundary Options: The following options are available:

    • Show Entire Boundary: Shows boundary lines on all boundary edges of the selected zone(s)

    • Show Boundary Line: Shows the boundary line closest to the selected point.

    • Hide Boundary Line: Hides the boundary line that is closest to the selected point.

    • Show Only Nearest Boundary Line: Shows only the boundary line that is closest to the selected point and hides all others.

 

Also see the I-Indx Bndy, J-Indx Bndy, and K-Indx Bndy buttons on the Boundary page of the Zone Style dialog.
