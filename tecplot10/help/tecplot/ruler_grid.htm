Ruler/Grid

The Ruler/Grid dialog controls the display of the workspace ruler and the paper grid.

 

The following options are available:

  • Show Paper on Screen: Select this check box to show the image of the paper in the workspace area. The paper image is displayed only if this option is selected.

  • Show Grid: Select this check box to show the paper grid. The grid is displayed only if this check box is selected and the Show Background check box is selected; the grid is displayed on the paper background.

  • Show Ruler: Select this check box to display the workspace paper rulers, which extend vertically down the right side of the workspace and horizontally across the bottom.

  • Grid Spacing: Select the grid spacing from among the choices in the drop-down. The grid spacing units are in centimeters, inches or points. The choices available are:

    • 0.1, 0.5, 1 (default), and 2 centimeters.

    • 0.1, 0.25, 0.5, and 1 inch.

    • 10, 24, 36, 50 points.

The units of the spacing chosen determine the units displayed in the Dimensions region, whether the grid is displayed or not.

  • Ruler Spacing: Select the spacing of numbers along the rulers from among the choices in the drop-down. The ruler spacing units are in centimeters, inches or points. The choices available are:

    • 1 (default) and 2 centimeters.

    • 1 inch.

    • 50 and 72 points.

The units of the spacing chosen determine the units displayed in the Dimensions region, whether the ruler is displayed or not.
