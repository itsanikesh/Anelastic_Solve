Alter Smooth

Use the Smooth dialog to smooth the values of a variable. Smoothing reduces "noise" and lessens discontinuities in data. Smoothing can also be used after inverse-distance interpolation to reduce the artificial peaks and plateaus.

Smoothing is applied to XY or Polar Lines in 2-D, across a 3-D surface, or in a 3-D volume depending on the state of the active window and the type of zone structure. Each pass of smoothing shifts the value of a variable at a data point towards an average of the values at its neighboring data points.

 

The following options are available:

  • Zone: Select the zone to smooth from the drop-down. For the best results, your zone should not intersect itself.

  • Variable: Select the variable to smooth from the drop-down. In 2-D, the variable cannot be assigned to either the X-axis or the Y-axis (that is, you cannot smooth X or Y). If the window is in 3-D, and you select the variable assigned to the Z-axis as the variable to smooth, Tecplot will smooth the variable, but treat the problem as 2-D (smoothing Z as a function over X and Y); otherwise 3-D data is smoothed in all three axis directions.

  • Number of Passes: Enter the number of smoothing passes to perform. The default is one. A greater number of passes will take more time but smooth the data more.

  • Coefficient: Specify the relaxation factor for each pass of smoothing, a number between zero and one (exclusively). Small numbers indicate small amounts of smoothing per pass. Numbers up to one indicate greater smoothing. The default is 0.5.

  • Boundary: Select the boundary conditions by which to smooth from the drop-down.

 

The following options are available (The default is Fixed):

  • Fixed: The points at the boundary are not changed in value. For finite-element zones, only fixed boundary conditions may be used.

  • First Order: The points at the boundary are smoothed based on the assumption that the first derivative normal to the boundary is constant. This will tend to cause contour lines of the smoothed variable to be perpendicular at the boundary.

  • Second Order: The points at the boundary are smoothed based on the assumption that the second derivative normal to the boundary is constant. This option results in the data being smooth near the boundary.

 

To smooth your data:

  1. Select the zone and variable.

  2. Check the Number of Passes, Coefficient, and Boundary parameters.

  3. Click Compute.

The Coefficient parameter specifies the amount of smoothing done per pass. Large numbers in Coefficient flatten peaks and noise quickly. Small numbers smooth less each pass, but the smoothing is more encompassing, rounding out peaks and valleys rather than eliminating them.

 

Smoothing has some limitations:

  • Finite-element zones can only be smoothed with Fixed boundary conditions.

  • Tecplot uses the current frame's axis assignments to determine the variables to use for the coordinates in the smoothing, and also to determine whether the smoothing should be done with XY Line, Polar Line, 2- or 3-D. Be careful if you have multiple frames with different variable assignments for the same data set.

  • Any axis scaling is ignored during smoothing.

  • For I-ordered zones or finite-element line segment zones, the current plot type can be XY Line, Polar Line, 2D Cartesian, or 3D Cartesian. In XY or Polar Line, the variable must be the dependent variable of one active mapping in one zone.

  • For IJ-ordered, finite-element triangle, or finite-element quadrilateral zones the current frame can be the 2D or 3D Cartesian plot type. You cannot smooth the variables assigned to the X- and Y-axes in 2-D.

  • For IJK-ordered finite-element triangle, or finite-element quadrilateral zones, you cannot smooth the variables assigned to the X-, Y-, and Z-axes. The IJK-mode is ignored. The zone is smoothed with respect to the entire 3-D volume.

  • Smoothing does not extend across zone boundaries. If you use a boundary condition option other than Fixed (such that values along the zone boundary change), contour lines can be discontinuous at the zone boundaries. Smoothing is performed on all nodes of a zone, and disregards value-blanking if it is active.
