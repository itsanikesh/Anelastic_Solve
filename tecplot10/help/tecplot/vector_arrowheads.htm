Vector Arrowheads

Use the Vector Arrowheads dialog to specify the size and angle of the vector arrowheads.

 

The following options are available:

  • Angle (deg): Specify the angle between the vector and the arrowhead. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 8, 10, 12, 14, 16, 18, 20, and 22.

  • Set Size Based On: Specify how the size of the vector arrowhead is determined, choosing one of the following options:

  • Fraction of Length: Enter a decimal value expressing the ratio of head length to body length. For example, 0.5 generates vectors with arrowheads one-half the length of the body of the vector.

  • Frame Units (%): Enter a percentage of the frame width to be used as the length for all arrowheads, regardless of vector size. This is useful for showing the vector direction even in regions where the magnitude of the vector issmall.
