3D Axis Reset

Resets the 3-D axes back to the system defaults.

 

The following options are available:

  • Origin: The 3-D axis origin reverts to the data center.

  • Axes Box (the box around the data, if displayed): Resizes the axes box to fully surround, with an extra margin, the plotted data. The extra margin is the default padding percentage added to provide extra space in the box around the data.

 

Note: This command does not change the current view, or the current magnification. Use the commands under the View or Workspace menus to change current view, or the current magnification to what you want.
