Reference Vector

Specifies whether, and how, a reference vector is displayed on your plot. The purpose of the reference vector is to present an absolute standard vector to which other vectors in the plot can be compared.

 

The following options are available:

  • Show Reference Vector: Select this check box to display the reference vector.

  • Origin (%): Enter coordinates of the starting point of the reference vector, as a percentage of frame width (X) and frame height (Y).

  • Color: Select the box outline color from the Select Colors dialog.

  • Angle (deg): Specify the orientation of the reference vector relative to the axis. The angle is measured in degrees counter-clockwise from horizontal. Horizontal numbers are at 0 degrees; vertical numbers are at 90 degrees.

  • Line Thickness: Specify the line thickness for the reference vector as a percentage of the frame width. You can select the Enter option and enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, 1.5.

 

You can alter these default values from the Size Preferences dialog.

  • Magnitude: Enter the size of the reference vector, in grid units.

  • Text Color: Choose the color for the text from the Select Color dialog.

  • Font: Choose the font for the text from the drop-down.

  • Number Format: Specify the number format on the Specify Number Format dialog which appears.

  • Offset (%): Specify the origin of the reference vector as a percentage of the frame width for X-axis, and frame height for Y-axis. Enter a value between zero and 100 in the text fields labeled X and Y.
