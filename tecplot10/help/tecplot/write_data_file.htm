Write Data File

Use this dialog to specify a file name and directory path with which to save your current data set.

 

To specify a file name either type the file name in the File Name text field or select an existing file name from those listed.

 

To change the file names listed on Motif:

  1. Choose a different directory (using the Directories list box).

  2. Specify a different file filter in the Filter (Name Search) field.

 

To change the file names listed in Windows:

  1. Select a different directory from the Save in drop-down.

  2. Select All files (*.*) in the Save as type field.

 

In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. For example: ftp://ftp.microsoft.com/myplot.plt.

 

In Motif, you may save a file with a URL by entering the full URL path in the Selection text field.
