Color Map

The Color Map dialog controls the colors displayed in flooded contour plots or multicolored mesh, scatter or vector plots. In this dialog you can choose from between five predefined color maps schemes, along with a sixth map scheme that interpolates between two colors – a user-specified minimum color and a user-specified maximum color. You also have two choices for defining your own color maps, either interactively by modifying the current color map, or by defining a color map file from scratch.

 

The following options are available:

  • Base Color Map: Selecting this button at the very top of the dialog allows you to choose a color map from among a selection of color maps schemes presented in the drop-down to be displayed in the color bar located below this option. The following options are presented in the drop-down:

  • Small Rainbow (default): Select this option to load in a simple, five color rainbow spectrum that ranges from blue to cyan to green to yellow and finally to red. RGB values can then be modified interactively.

  • Large Rainbow: Selecting this option loads in a seven color rainbow spectrum that ranges from blue to cyan to green to yellow to red to purple and finally to white. RGB values can then be modified interactively.

  • Modern: Load in a spectrum of seven distinct colors that change in intensity from dark to light within each color. The colors presented in this scheme are dark purple to light purple, dark blue to light blue, and so on for cyan, green, yellow, orange, and red. RGB values can then be modified interactively.

  • Gray Scale: Loads in a black to white, gray scale spectrum. RGB values can then be modified interactively.

  • Wild: Loads in a random color collection color map that uses random RGB values. Wild is different each time you select it. So, you can select this options several times until a combination is displayed that you like.

  • Two Color: To specify two colors for your spectrum (start and end colors); Tecplot then interpolates a range of colors between these two values.

  • User-Defined: Load in and modify one of the standard color maps. For information on modifying a color map see Modifying User-Defined Color Maps.

  • Raw User-Defined: Allows you to load in one of the standard color maps into the Color Map dialog color bar. You may then save this map off to a file. Use the Copy Color Map to File option, under the Workspace menu, to save this color map to a color map file. The file that is saved from this option lists all the RGB values for the whole spectrum displayed, not just the RGB values for the control points like the files that the User-Defined option saves. You can then go into this file and modify the values by hand. This means you can insert new colors, or other effects, to customize you own color map from scratch. Use the Paste Color Map from File option, under the Workspace menu, to load your customized color map as the current color map. The customized color map will be shown in the color bar at the top of the Color Map dialog, and the Base Color Map option shows "Raw User-Defined.”

 

To load in one of the standard color maps, after selecting the User-Defined option or Raw User-Defined:

  1. Click on the Copy Standard Color Map button at the bottom of the Color Map dialog.

  2. Choose one of the choices in the drop-down. The choices available are:

    • Small Rainbow.

    • Large Rainbow.

    • Modern.

    • Gray Scale.

    • Color Bar: This is the color spectrum display area just below the Base Color Map option at the top of the Color Map dialog. It shows the color mapping that is currently active along with any control points for that mapping. A color mapping is a linear variation of colors that represent the minimum to maximum contour value in a flooded contour plot.

    • RGB Values for Control Points: Shows the control point that is currently active (it is highlight in black). The control points are numbered from one to the maximum number set for this color map. Changing the RGB values changes the color that is displayed on the left or the right side of a control point.

    • Left RGB: The RGB values for the left side of a control point. There is one slider for setting the value for each color value. The RGB values range from zero (off) to 255 (full). The combination of values from each of these colors are combined to create colors in the spectrum. Black is 0-0-0; white 255-255-255. The remainder of the colors are in between these values.

    • Red, Green, and Blue: Use the sliders to specify the RGB the minimum (maximum) color values of the left RGB color currently being edited for the active control point. Adjust the slider by clicking-and-dragging.

 

To use fine control to increase or decrease the displayed value:

  1. Click on the slider.

  2. Use the left and right arrow keys.

    • Right RGB: The RGB values for the right side of a control point. There is one slider for setting the value for each color value. The RGB values range from zero (off) to 255 (full). The combination of values from each of these colors are combined to create colors in the spectrum. Black is 0-0-0; white 255-255-255 The remainder of the colors are in between these values.

    • Red, Green, and Blue: Use the sliders to specify the RGB the minimum (maximum) color values of the right RGB color currently being edited for the active control point. Adjust the slider by clicking-and-dragging.

 

To use fine control to increase or decrease the displayed value:

  1. Click on the slider.

  2. Drag the slider to the right or left.

  3. Use the left and right arrow keys.

    • Right RGB same as Left: Selecting this check box gives a continuous spectrum value across a control point, that is, the value on the left side is the same on the right side. When this is not selected, the right-hand RGB value can jump up to a completely different color value. This results in a stepped color effect, like that seen in the Modern color map.

    • Number of Control Points: This allows you to increase or decrease the number of control points used in a color map. The number of control points can be any integer from two to nine. Added control points will be placed on the right end of the color spectrum. This option is only available user-defined color maps.

    • Copy Standard Color Map: This copies a specific standard color map into the color bar when the Base Color Map option currently selected is either User-Defined or Raw User-Defined.

 

To load in one of the standard color maps:

  1. Select either User-Defined, or Raw User-Defined option from the Base Color Map drop-down.

  2. Click on the Copy Standard Color Map button at the bottom of the Color Map dialog.

  3. Choose one of the choices in the drop-down. The choices available are:

    • Small Rainbow.

    • Large Rainbow.

    • Modern.

    • Wild.

 

The color map option you choose is displayed in the color bar at the top of the Color Map dialog.

  • Redistribute Control Points: Selecting this button moves the control points back to their default positions without resetting any RGB values to their default values. This means that any changes you made to the RGB values remain unchanged while the control points are redistributed equally along your customized spectrum.

  • Reset: Selecting this button results in two actions. Reset moves the control points back to their default positions and resets all the RGB values to their default values. Use this to put a color map to its original values.
