Preferences Colors

Use the Preferences menu to select options for setting the control size, color, and other Tecplot preferences.

  • Colors: Selecting this option calls up the Color Preferences dialog, from which you set the RGB values for the basic colors in Tecplot.

  • To change a color, click on it in the palette and alter its RGB values with the sliders. You may alter multiple colors by selecting those colors and changing their RGB values.

  • Choosing Reset Selected Color or Reset All Colors will restore the default RGB values.

  • All color changes take effect when you click OK.
