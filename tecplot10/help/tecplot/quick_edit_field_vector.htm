Quick Edit Field Vector

This area controls whether vectors are displayed for selected zones, and if so, using what plot type.

  • Vect Show: Display vectors for the selected zones. This option allows you to turn off just the Vector zone layer for specific zones, without completely deactivating the zones.

    • Yes: Show the vectors for the selected zones. If the Vector zone layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate the vector layer.

    • No: Turn off the vectors for the selected zones.

  • Vect Type: Choose the vector type for the selected zones. The following options are available:

    • Tail at Point: Display regular vectors – a simple stick vector with length proportional to the local velocity magnitude (the square root of the sum of the squares of the vector components). The tail of the vector is positioned at the data point.

    • Head at Point: Display regular vectors – a simple stick vector with length proportional to the local velocity magnitude (the square root of the sum of the squares of the vector components). The head of the vector is positioned at the data point.

    • Anchor at Midpoint: Display regular vectors – a simple stick vector with length proportional to the local velocity magnitude (the square root of the sum of the squares of the vector components). The midpoint of the vector is positioned at the data point.

    • Head Only: Display vectors as heads only, without the vector shaft.
