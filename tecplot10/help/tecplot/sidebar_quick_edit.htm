Sidebar Quick Edit

Use the Quick Edit dialog to make quick changes to objects currently selected in the workspace, and to make global plot attribute changes. The top section of the Quick Edit dialog contains options for the plot layers; the contents of this section depends upon the current plot type.

 

Field Plot Layer Options

These Quick Edit options, located at the top section in the dialog, apply to 2- and 3-D field plot layers and appear only when the current plot type is 2D or 3D Cartesian.

  • Mesh Options.

  • Contour Options.

  • Vector Options.

  • Scatter Options.

  • Shade Options.

  • Boundary Options.

 

Line-Plot Layer Options

These Quick Edit options, located at the top section in the dialog, apply to plot layers and appear only when the current plot type is XY Line.

  • Lines Options.

  • Symbols Options.

  • Bars Options.

  • Error Bars Options.

 

General Options

These Quick Edit options apply specific plot attributes to the selected portion of the plot:

  • Color Options.

  • Symbol Options.

  • Line Options.

  • Polyline Arrow Option.

  • Arrowhead Options.

  • Font Options.

  • Alignment Options.
