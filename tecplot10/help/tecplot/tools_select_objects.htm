Tools Select Objects

Select the Selector tool to use the cursor to select objects in your workspace. In the Selector mode, you can select objects by pointing to them.

 

To select an object and open that object's attributes dialog either double-click on any object, or drag the cursor to select groups of objects (calls up Group Select dialog). Click OK, then Object Details.

 

The following objects may be selected:

  • Frames.

  • Axes.

  • Text.

  • Geometries (polylines, circles, ellipses, squares, rectangles, and images).• Zones.

  • Viewport/Grid Area.

  • Axis Titles.

  • Contour Labels.

  • Streamtraces.

  • Streamtrace Termination Lines.

  • Legends.

  • 3D Orientation Axis.

 

All of these but Legends, Termination Lines, and the 3D Frame Axis, can be selected by a group select, in which you drag the cursor to define a rubber band box specifying an area in which to select all objects. The Group Select dialog appears, asking you which objects within the rubber band box to select. When the object is selected, the object displays selection handles. Only objects completely within the rubber band box are selected.

 

Selected objects can be operated upon by the Quick Edit dialog, or can be manipulated to some extent by the Selector itself.

 

The following objects can be moved (translated) by the Selector:

  • Frames.

  • Axis Grid Area.

  • Text.

  • Geometries.

  • Contour Labels.

  • Streamtraces.

  • Streamtrace Termination Lines.

  • Legends.

  • 3D Frame Axis.

 

In particular, note that zones may not be moved. Only the start position of a streamtrace may be moved; the streamtrace itself is recalculated when you drop the start position in its new location.

 

To move a selected object:

  1. Move the cursor through the rectangle outlined by the selection handles until the cursor changes to the translate cursor.

  2. Drag the selected object to the desired location.

  3. In addition to moving objects, you can sometimes use the Selector to resize objects.

 

The following objects may be resized by the Selector:

  • Frames.

  • Axis Grid Area.

  • Rectangle and Image Geometries.

 

To resize a selected object:

  1. Move the cursor over one of the selection handles until the cursor changes to the resize cursor.

  2. Drag the resize cursor until the object is the desired size and shape. The top and bottom resize handles adjust only the object's height; left and right handles adjust only the object's width. Resize corners adjust both height and width.
