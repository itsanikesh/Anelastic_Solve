Enter Mapping Name

Use the Enter Mapping Name dialog to enter or construct a name for the selected mappings.

 

You can simply type a map name in the Map Name text field, or you can build a "name template" by using the following Inserts buttons:

  • Zone Name: Select this to add the insert "&ZN&" to the Map Name text field; this is interpolated to the actual zone name when displayed in the Map Name field of the Plot Attributes dialog.

  • Zone Number: Ads the string "&Z#&" to the Map Name field, which is then replaced with the actual number of the zone assigned to the mapping.

  • Map Number: Adds the string "&M#&" to the Map Name field, which is then replaced with the actual number of the mapping.

  • Independent Var: Adds the string "&IV&" to the Map Name field, which is then replaced with the actual name of the independent variable assigned to that mapping.

  • Independent Var Number: Adds the string "&I#&" to the Map Name field, which is then replaced with the actual name of the dependent variable assigned to the mapping.

  • Dependent Var: Select this to add the insert "&DV&" to the Map Name text field; this is interpolated to the actual dependent variable name when displayed in the Map Name field of the Plot Attributes dialog.

  • Dependent Var Number: Adds the string "&D#&" to the Map Name field, which is then replaced with the actual number of the dependent variable assigned to the mapping.

  • X-Axis Num: Select this to add the insert "&X#&" to the Map Name field; this is interpolated to the actual X-axis number (that is, an integer from one to five) when displayed in the Map Name field of the Plot Attributes dialog.

  • Y-Axis Num: Select this to add the insert "&Y#&" to the Map Name field; this is interpolated to the actual Y-axis number (that is, an integer from one to five) when displayed in the Map Name field of the Plot Attributes dialog.

 

You can also enter text and select inserts (or simply type the insert directly into the text field). For example, you could type "My Map &DV&" and have all the selected map names start with the phrase "My Map" and end with the Dependent Variable name.
