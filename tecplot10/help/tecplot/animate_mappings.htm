Animate Mappings

Use the Animate Mappings dialog to display all or a specified subset of the mappings in the current data set, one at a time. You specify a start map (the first map you want displayed), an end map, and a map skip to use in the animation sequence.

 

The following options are available:

  • Start Map: Select the initial map to display from the drop-down of the data set's mappings.

  • End Map: Select the last map to display from the drop-down of the data set's mappings.

  • Map Skip: Enter the map skip.

 

Use one to display all maps from Start Map to End Map. Use two to display every other map from Start Map to End Map. Use three to display every third map, and so on.

  • Current Map: Shows the current map displayed. Use the up or down arrows to manually step through the maps.

  • Animate: There are three options: On Screen, to AVI file, to RM file. If On Screen is selected the animation will occur on the screen only. If AVI or RM are selected, the appropriate Exporter dialog appears when you click Animate. This is followed by the file dialog to specify the file name and path for the movie file. Finally, the animation will be sent to the file only and not appear on the screen.
