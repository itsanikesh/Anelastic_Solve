Probe At Index

Use the Probe At dialog to examine variable values or zone and cell information for any point in the data set. You can probe at either a spatial position, or by cell indices.

 

The Probe At dialog has two pages: Position and Index.

 

Position

  • Enter Coordinates (Probe At Position only): Enter the X-, Y- and if 3-D the Z-coordinates of the probe location. For each coordinate, you can use the up or down buttons to increase or decrease the coordinate.

  • Probe within Volume: Select this check box to have Tecplot use the X-, Y- and Z-coordinates you entered for a probe location literally within the data volume. When this check box is not used, Tecplot transforms the X-, Y- and Z-coordinates you entered for a probe location from data units into frame units and searches behind it to find the data at that point. This means that it is better to leave this option off, and let Tecplot find the closest position for information, such as a plane wing surface, which is difficult to probe its surface directly by entering a coordinate.

  • Do Probe: Perform the probe operation, and display the results in the Probe dialog.

  • Index

  • Zone: Select from the zone into which you want to probe. Then enter the I-, J- and if the data set is 3-D, the K-indices of the probe location.

  • Do Probe: Perform the probe operation, and display the results in the Probe dialog.
