Addon General Text Loader Titles

This page deals with options for General Text Loader titles.

  • Use Title: If checked, you can enter the data set title here, rather than have General Text Loader scan the file for it.

  • Use line number: If checked, you can enter the line number of the file to look for the data set title. Unless the delimiter is specified as fixed, General Text Loader skips white space on the line until text, and then reads until the delimiter indicated is found. To include spaces in the title, enclose it in double quotes.

  • Use keyword: If checked, you can enter a keyword which indicates the data set title line. The title will be read first line containing this keyword (case insensitive).

 

General Text Loader searches for a title on this line in the following order, unless the delimiter is specified as fixed:

  1. First, it will look for any text enclosed in double quotes. If it finds this, then the enclosed text will be read as the title.

  2. If no text in double quotes is found, then the first non-white space text after the keyword, ending with the indicated delimiter, will be used.

  • Text Delimiter: The text delimiter indicates when the end of text has been reached. You can set it to one of the following:

Auto: Space, tab, comma, semicolon.

Tab.

Comma.

Semicolon.

  • Fixed: Each width number of characters on the line is a token field. White space is removed from the beginning and end of the field.

  • Width: If the delimiter is fixed, enter the width of each field here.
