Print Setup Windows

Use the Windows Print Setup dialog to choose options for printing with your Windows printers.

 

The following options are available:

 

Printer

Specify the printer and set its properties.

  • Name: Indicates your default printer. Another installed printer may be selected from the drop-down.

 

Paper

Specify the paper size and source tray using the following drop-downs:

  • Size: Choose paper size from the drop-down. The available choices are printer-dependent. Paper size can also be modified in the Print Setup dialog. If you make a change in one dialog, Tecplot immediately reflects the change in the other dialog.

  • Source: Choose a paper tray from the drop-down. The default, Print Manager Setting, uses the paper tray chosen in Print Manager.

 

Orientation

Specify the orientation for the printed plot using one of the following options:

  • Portrait: To select Portrait orientation, in which the horizontal axis of the plot is aligned with the short side of the paper.

  • Landscape: To select Landscape orientation, in which the vertical axis of the plot is aligned with the short side of the paper.

 

Orientation settings can also be modified in the Paper Setup dialog. If you make a change in one dialog, Tecplot immediately reflects the change on the other dialog.

 

By default, Tecplot uses the standard Windows print drivers. You can choose to use Tecplot's native print drivers, in which case you set up printing using the Tecplot-specific Print Setup dialog.

 

To use the native Tecplot drivers, add the following line to your tecplot.cfg file:

 

$!INTERFACE USETECPLOTPRINTDRIVERS=YES
