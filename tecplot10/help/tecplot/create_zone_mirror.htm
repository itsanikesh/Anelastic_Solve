Create Zone Mirror

Use the Create Mirror Zone dialog to create a mirror-image replica of one or more existing zones.

 

The following options are available:

  • Mirror Axis: Select the check box for the desired mirror:

  • In 2-D the desired mirror axis: X or Y.

  • In 3-D the desired mirror plane: XY, XZ, or YZ.

  • Source Zone(s): Select zones for transformation. The default is no zones selected.

 

To select a single zone, click on it. To select a range of zones:

  1. Click on the first or last zone in the range.

  2. Shift-click on the last or first zone in the range.

 

To select an arbitrary set of zones:

  1. Click on the first desired zone.

  2. Ctrl-click on subsequent zones until all desired zones are selected.

  3. To create the zone, click Create.
