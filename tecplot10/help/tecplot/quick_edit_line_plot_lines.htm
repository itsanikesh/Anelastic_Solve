Quick Edit Line Plot Lines

This area controls whether a line plot is provided for selected mappings, and if so, which type.

 

The following options are available:

  • Line Show: Display line plots for the selected maps. This option allows you to turn off just the Lines map layer for specific maps, without completely deactivating the maps. The following options are available:

    • Yes: Show the line plots for the selected maps. If the Lines map layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate the Lines layer.

    • No: Turn off the line plots for the selected maps.

  • Line Type: Choose a line type from the drop-down of the following choices:

    • Line Segment: Tecplot connects all data points with a single polyline, using a straight line segment to connect point 1 with point 2, point 2 with point 3, and so on.

    • Polynomial Fit: Tecplot fits a polynomial curve to the data points. Use the Curves page to specify the degree of the polynomial.

    • Exponential Fit: Tecplot fits an exponential curve to the data points. To use this line type, Y-values for the variable must be all positive or all negative.

    • Power Fit: Tecplot fits a curve of the form Y=a*X**b to the data points. To use this curve type, Y-values for this variable must be all positive or all negative, and the X-values must all be positive.

    • Spline: Tecplot fits a spline (a special type of smooth curve) through every data point. This curve type assumes that the points of the dependent variable represent functional values of the independent variable. That is, a spline is drawn through the data points after sorting in increasing values along the independent variable axis. The spline is a single-valued function of the independent variable. Use the Curves page of the Mapping Style dialog to specify whether the spline is clamped or unclamped.

    • Para Spline: Tecplot fits a spline under the assumption that both X and Y are functions of some other independent variables, where s is the accumulated scaled distance between points (that is, (x,y)=F(s)). No sorting of the points is performed; the order of the data points from the data file is used.

    • Extended: Tecplot uses an add-on to fit a curve to your data.
