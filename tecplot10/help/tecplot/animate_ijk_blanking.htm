Animate ijk blanking

Use the Animate IJK_Blanking dialog to animate a sequence of Tecplot renderings starting with an initial set of blanked IJK-indices and proceeding in a series of interpolated steps to a final set of blanked IJK-indices.

 

Before you can animate you must turn on the IJK-Blanking and specify which zone you want to perform blanking in the IJK-Blanking dialog. You also must specify in the animate IJK-Blanking dialog the initial and final blanking parameters, and the number of steps to use in the animation sequence. The number of steps must be greater than or equal to two; use two to display only the initial and final blanking configurations.

 

The following options are available:

  • Starting Index (% of Max): Specify the initial blanking configuration, by entering minimum and maximum I-, J- and K-indices for the chosen zone.

  • Ending Index (% of Max): Specify the final blanking configuration by entering minimum and maximum I-, J- and K-indices for the chosen zone.

  • Number of Steps: Enter the number of steps required to move from the initial configuration to the final configuration. This number must be at least two; the value two specifies that Tecplot shows only the initial and final configuration. Tecplot interpolates a set of intermediate configurations for each additional step.

  • Current Step: Shows the current step displayed. Use the up or down arrows to manually step through the steps.

  • Animate: There are three options: On Screen, to AVI file, to RM file. If On Screen is selected the animation will occur on the screen only. If AVI or RM are selected, the appropriate Exporter dialog appears when you click Animate. This is followed by the file dialog to specify the file name and path for the movie file. Finally, the animation will be sent to the file only and not appear on the screen.
