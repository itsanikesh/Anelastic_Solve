Sidebar Mapping Style

Select Mapping Style from the sidebar to call up whichever dialog most closely reflects the current state of the cursor.

 

If the current tool is either the Selector or Adjustor, the button is labeled Object Details. If an object is selected, the dialog controlling that object's attributes is displayed when you click Object Details. If any other tool is selected, the button says Tool Details.

 

For example, if you select a legend and then click Object Details, the appropriate legend dialog appears. If no object is selected, and the Selector or Adjustor tools are active, a page of the Mapping Style dialog appears, with the appropriate page determined as follows: if the dialog has previously been opened to a particular page, the dialog opens at that page, unless, in the meantime, you have added one or more map layers to the plot, in which case the dialog opens to the page for the layer most recently added.

 

There are seven pages on the Mapping Style dialog for the XY Line plot type, five of which apply to the Polar Line plot type. They are:

  • Definitions.

  • Lines.

  • Curves.

  • Symbols.

  • Error Bars.

  • Bars.

  • Indices.
