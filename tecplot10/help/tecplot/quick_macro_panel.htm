Quick Macro Panel

Use the Quick Macro Panel to access any quick macros you may have defined.

 

A quick macro is any macro function defined inside the file tecplot.mcr (or any other file you might specify). When you start Tecplot, it looks for the file tecplot.mcr in the Tecplot Home Directory. If the file exists, the names of the macro functions defined in that file are placed on the buttons in the Quick Macro Panel.

 

To specify an alternative quick macro file, use the -qm flag on the Tecplot command line. For example, the following command runs Tecplot and installs the macro functions in the file myteccmd.mcr on the Quick Macro Panel:

tecplot -qm myteccmd.mcr

 

To have the Quick Macro Panel appear at the startup of Tecplot, use the -showpanel flag on the Tecplot command line. For example, the following command runs Tecplot and calls up the Quick Macro Panel with the file myteccmd.mcr.

 

tecplot -qm myteccmd.mcr -showpanel

 

You can define as many quick macros as you like. The Quick Macro Panel has fifteen buttons on each page, but creates additional pages as needed to accommodate additional macros.

 

Use the Previous and Next buttons to access additional pages of the Quick Macro Panel.
