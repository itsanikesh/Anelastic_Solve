Iso Surface Details Definition

Use the Iso-Surface Details dialog to control Tecplot’s rendering of iso-surfaces, which are based on a contour variable.

 

The following options are available:

  • Show Iso-Surfaces: Select this check box to display iso-surfaces.

  • Define Iso-Surfaces Using: Use the drop-down to select the approrpriate contour group.

  • Draw Iso-Surfaces at: Use this drop-down to have Tecplot draw iso-surfaces at each contour level, or at the value(s) you determine. This is available only when Iso-surfaces is selected.

  • Value 1: Enter a value in the text field, or use the arrows at the right to increase or decrease the value.

  • Value 2: Enter a value in the text field, or use the arrows at the right to increase or decrease the value.

  • Value 3: Enter a value in the text field, or use the arrows at the right to increase or decrease the value.

  • Contour Variable Min: Indicates the minimumcontour variable calculated by Tecplot.

  • Contour Variable Max: Indicates the maximum contour variable calculated by Tecplot.
