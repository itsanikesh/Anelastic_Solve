Tools Twist Rotation

Select the Twist rotation tool to rotate the plot about the eye-origin ray.

 

To rotate the plot about the eye-origin ray (that is, rotate in the plane of the screen), drag the cursor across the screen.

 

To rotate the current plot view counter-clockwise, move the cursor counter-clockwise.

 

To rotate the current plot view clockwise, move the cursor clockwise.

 

Special Mouse Commands

  • Ctrl-Right-Click: Smooth zoom in and out of the data.

  • Middle-Click: Smooth zoom in and out of the data.

  • Right-Click: Translate the data.
