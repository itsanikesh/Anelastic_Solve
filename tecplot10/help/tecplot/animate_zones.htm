Animate Zones

Use the Animate Zones dialog to display all or a specified subset of the zones in the current data set, one at a time. You specify a start zone (the first zone you want displayed), an end zone, and a zone skip to use in the animation sequence.

 

The following options are available:

  • Start Zone: Select the initial zone to display from the drop-down of the data set's zones.

  • End Zone: Select the last zone to display from the drop-down of the data set's zones.

  • Zone Skip: Enter the zone skip number. Use one to display all zones from Start Zone to End Zone. Use two to display every other zone from Start Zone to End Zone. Use three to display every third zone, and so on.

  • Current Zone: Shows the current zone displayed. Use the up or down arrows to manually step through the zones.

  • Animate: There are three options: On Screen, to AVI file, to RM file. If On Screen is selected the animation will occur on the screen only. If AVI or RM are selected, the appropriate Exporter dialog appears when you click Animate. This is followed by the file dialog to specify the file name and path for the movie file. Finally, the animation will be sent to the file only and not appear on the screen.
