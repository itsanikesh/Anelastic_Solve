Tools Spherical Rotation

Select the Spherical rotation tool to rotate the object about the Z-axis.

 

To rotate the object horizontally about the Z-axis, move the cursor horizontally across the plot.

 

To tilt the Z-axis vertically, (that is, control the tilt of the Z-axis relative to the eye/origin ray), move the cursor vertically across the plot.

 

Special Mouse Commands

  • Ctrl-Right-Click: Smooth zoom in and out of the data.

  • Middle-Click: Smooth zoom in and out of the data.

  • Right-Click: Translate the data.
