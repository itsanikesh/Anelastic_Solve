Import PLY Polygon Files

Use this loader to load 3-D triangular surface files with the .ply extension.

 

This format is often used to store surfaces generated from tessellation of 3-D range measurement data. Files may be either ASCII or binary, but must contain both vertex and face elements (sections).

 

This loader is freely available from the Tecplot Web site at http://www.tecplot.com.
