Animate Streamtraces

Use the Animate Streamtraces dialog to create animated images of streamtraces.

 

You must have streamtrace timing turned on (either markers or dashes) before you animate streamtraces. You specify the number of images shown for each streamtrace cycle, and the number of cycles to show in the animation sequence. The resulting animation shows streamtrace markers at each step of the animation, "moving" down the streamtrace.

 

The following options are available:

  • Steps Per Cycle: Enter the number of steps per cycle for the animation.

  • Number of Cycles: Enter the number of cycles to display.

  • Animate: There are three options: On Screen, to AVI file, to RM file. If On Screen is selected the animation will occur on the screen only. If AVI or RM are selected, the appropriate Exporter dialog appears when you click Animate. This is followed by the file dialog to specify the file name and path for the movie file. Finally, the animation will be sent to the file only and not appear on the screen.
