Import Fluent

Fluent Data Loader allows you to read Fluent Version 5 or 6 case (.cas) and data (.dat) files into Tecplot.

 

To load files from earlier versions of Fluent, you must first import them into Fluent 5 or 6, then save them as Fluent 5 or 6 files. If a particular variable is present in a data file for only one zone, it is set to zero in Tecplot for all other zones.

 

The following options are available:

  • Load Case and Data Files: Loads both a case and a data file. The grid comes from the case file, and the solution comes from the data file. Specify the case and data files below.

  • Load Case File Only: Loads the grid from a case file. Specify the case below.

  • Load Residuals Only: Loads the residual data (convergence history) from a data file. The residuals are not scaled or normalized.

  • Load Multiple Case and Data Files: Displays the File List region of the dialog. You may load either one case file and multiple matching data files, or pairs of matching case and data files. Enter all files in the list below.

  • Case File: Indicates the case (.cas) file from which the grid will be loaded. Type the name of the case file, or click the button and select the name of the file from the resulting dialog.

  • Data File: Indicates the data (.dat) file from which solution or residual data will be loaded. Type the name of the data file, or click the button and select the name of the file from the resulting dialog.

  • File List: Includes the list of case and data files and controls for selecting files for the Load Multiple Case and Data Files option. Also contains controls for indicating time levels for unsteady solutions.

  • Add Files: Displays the multiple file selection dialog to allow you to add files to the list of files to be loaded. Selected files will be added to the end of the list.

  • Remove: Removes selected files from the list.

  • Remove All: Clears the file list.

  • Flow Solution is Unsteady/Time Interval: For loading multiple case and data files, indicates that individual data files or case/data file pairs represent different time levels of an unsteady solution. The time value (starting at a value of 0) will be stored in auxiliary data for each zone. This information is not used by Tecplot, but may be used by other Tecplot add-ons. Refer to Tecplot’s User’s Manual for more information on auxiliary data.

  • Options: The Options portion of the dialog allows you to choose which zones and variables to load, and contains the following controls:

  • Load Cells and Boundaries: Loads all zones from the case file. Each fluid or solid cell zone and each boundary zone will be displayed as a separate zone in Tecplot.

  • Load Cells Only: Loads only the cell (solution) zones. Each zone will be displayed as a separate zone in Tecplot.

  • Load Boundaries Only: Loads only the boundary zones. Each zone will be displayed as a separate zone in Tecplot.

  • Select Zones and Variables to Load: Allows you to choose in a separate dialog which zones and variables to load. The selection dialog is displayed after you click OK.

  • Include Particle Data: Loads discrete particles, if any, from the data file. All particles from a particular injection will be displayed in a single Tecplot zone.

  • Averaging: By default, non-grid variables are stored at cell centers, consistent with Fluent. This section of the dialog contains controls that allow you to average the data to the nodes, which may be useful for other add-ons that cannot handle cell-centered data.

  • Average to Nodes: Directs the loader to average Fluent’s cell-centered data to the grid nodes. Choose the averaging method below.

  • Arithmetic: A simple, fast arithmetic averaging will be performed.

  • Laplacian: A more accurate, much slower averaging will be performed that accounts for hanging nodes and cell sizes.
