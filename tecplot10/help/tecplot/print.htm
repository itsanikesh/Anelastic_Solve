Print

Prints current plot to specified printer or file.

 

Tecplot supports the standard Windows printer drivers. You can, optionally, configure Tecplot to use the native Tecplot print drivers for the following formats: PostScript, HP-GL, and HP-GL/2.

 

The available options are:

  • Format: Indicates the current printer for the plot.

  • Send Output to File: Selecting this check box will send your output to a file instead of a printer.

  • Print Setup: Calls up the Print Setup dialog.

  • Render Options: Calls up the Print Render Options dialog.

  • Preview: Displays Tecplot’s print preview screen, where you will see a preview image of what will be printed.

 

Ctrl-P is the keyboard shortcut for the Print dialog.
