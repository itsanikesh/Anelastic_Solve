Contour Details Lines

All Contour Details dialogs include the following options:

  • C1, C2, C3, C4: Select which contour group is viewed and modified in the Contour Details dialog.

  • Var: Select the contour variable for the selected contour group.

  • More>>: Expand the Contour Details dialog to include additional options.

  • Less<<: Contract the Contour Details dialog to include only the contour group selection and the contour variable selection controls.

 

Use the Lines page of the Contour Details dialog to specify the contour line mode.

There are three options:

  • Use Zone Line Pattern: All lines use the zone line pattern specified in the Line Pttrn attribute on the Contour page of the Zone Style dialog.

  • Skip to Solid: Every nth line is dashed, where n is specified in the field Number of Dashed Lines to Draw between Solid. Use the field labeled Pattern Length for Dashed Lines (%) to specify the pattern length.

  • Dashed Negative Lines: Contour lines with positive values of the contour variable are drawn solid, those with negative values are dashed. Use the field labeled Pattern Length for Dashed Lines (%) to specify the pattern length.

 

The following fields may be enterable, depending on the choice of contour line mode:

  • Pattern Length for Dashed Lines (%) (Skip to Solid and Dashed Negative Lines only): Specify the pattern length for dashed lines as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.5, 0.8, 2, 3, and 5. You can alter these default values from the Size Preferences dialog.

  • Number of Dashed Lines to Draw between Solid Lines (Skip to Solid only): Enter the number of dashed lines to be drawn between each solid line.
