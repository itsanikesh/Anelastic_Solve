Quick Edit Line

This area controls the line pattern, pattern length, and line thickness for all selected objects.

 

The following options are available:

  • Line Pttrn: Choose the line pattern for the selected zones from among the following: Solid, Dashed, DashDot, Dotted, LongDash, DashDotDot.

  • Pttrn Lngth: Specify the pattern length for the selected line pattern, as a percentage of the frame width. You can choose Enter and enter an exact percentage in the Enter Value dialog that appears, or choose one of the values in the drop-down. The default values available are: 0.5, 0.8, 2, 3, and 5. You can alter these default values from the Size Preferences dialog.

  • Line Thck: Specify the line thickness for the vectors as a percentage of the frame width. You can choose Enter and enter an exact percentage in the Enter Value dialog that appears, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, and 1.5. You can alter these default values from the Size Preferences dialog.
