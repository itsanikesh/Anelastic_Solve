Save Layout As

Ctrl-W is the keyboard shortcut for the Save Layout As dialog.

 

Save Layout As saves your current data as a layout or layout package file. These are:

 

  1. Layout files: Store information about all the frames in the Tecplot workspace, including identification of the data used by each frame. The file extension is .lay

  2. Layout Package files: An extension to layout files where data and an optional preview image are included. The file extension is .lpk

 

To save a layout file, do the following:

 

  1. From the File menu select Save Layout.

  2. Choose Linked Data (*.lay) using the Save as type drop-down. By default, Tecplot saves the name of the data files used in the layout with their absolute file paths. If you want to save your layout using relative file paths, click the Use Relative Path check box.

  3. Specify a file name for the layout file. In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. For example ftp://ftp.microsoft.com/myplot.plt. Note that the two file types are mutually exclusive. You cannot browse disk files when the URL check box is selected, and you cannot enter a URL if the URL check box is not selected.

  4. To open (or save) the URL, click the Open URL (or Save URL) button. In Motif, you may open a file with a URL by entering the full URL path in the Selection text field. If you have changed a data set since you last read it in or wrote it out, Tecplot prompts you for a file name under which to save the changed data. If your layout has multiple data sets, Tecplot prompts you for a file name for each modified data set.

  5. Once you save a layout, the layout file name appears in the header of the Tecplot application window. To save your layout to this file again, simply choose Save Layout from the File menu.

 

To save a layout package file, do the following:

 

  1. From the File menu select Save Layout.

  2. Choose Packaged Data (*.lpk) from the Save as type drop-down.

  3. Choose whether or not you want a preview image included with your layout package file.

  4. Specify a file name for the layout package file.

 

To show a different set of file names in the Files list box, either choose a different directory using the Directories list box (Motif), or the Save in drop-down (Windows). You may specify a different file filter by entering a filter with wildcard symbols in the List Files of Type drop-down and File Name text fields (Motif), or the Save as type drop-down and File Name text field (Windows).
