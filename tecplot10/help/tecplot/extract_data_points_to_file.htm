Extract Data Points to File

Use the Extract Data Points to File dialog to specify a file name for points extracted from a data set, either as a discrete set, along a polyline, or from a selected polyline geometry.

 

To specify the new file name, either enter a configuration file name in the File Name list box, or select a file from the File Name list box.

 

You can show a different set of files in the File Name list box by choosing a different directory (using the Directories list box or Drives drop-down) or specifying a different file filter (using the List Files of Type drop-down).
