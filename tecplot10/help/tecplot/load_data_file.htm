Load Data File

Use the Load Data File dialog to read a single data file into the current Tecplot frame.

 

To read a file:

  1. Click on the file's name in the scrolled list labeled under the Look in label (Windows), or the Files label (Motif). In Windows, you may specify a URL by selecting the URL check box. When selected, you may enter a full URL as the file name. The URL must start with either ftp:// or http://. For example ftp://ftp.microsoft.com/myplot.plt. In Motif, you may open a file with a URL by entering the full URL path in the Selection text field. In Windows, you may load multiple files by using Ctrl-click to select additional files.

  2. Click OK, or Type the file name into the File Name text field and then click OK (Open in Windows). By default, this scrolled list shows data files in the current directory.

  3. To show a different list of files either click on a different directory in the Directories scrolled list (Motif) or the Look in drop-down (Windows). You may edit the name search filter by specifying a new path or file extension in the File Name field (Windows).

 

To bring up a dialog for selecting multiple data files:

Click the button labeled Multiple Files.

 

To select which zones and variables are read into Tecplot:

 

1. Select the check box labeled Specify Options.

2. Click OK.

3. Use the Load Data File Options dialog to specify which portions of the data set to read.
