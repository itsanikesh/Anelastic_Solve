Load Data File Options General

This page asks you to specify:

  • What type of data to load.

  • Field data (zone records; often your measured or calculated data).

  • Text, geometry, and/or custom label records.

  • I-, J-, and K-Skip factors (a value of one reads points of any I-index, a value of two reads points of every other I-index, and so on). For large data files, skip factors can greatly reduce the size of the data set within Tecplot, and hence speed processing.

  • The initial plot type (3D Cartesian, 2D Cartesian, XY Line, Polar Line, Sketch). By default, the initial plot type is 3D Cartesian for IJK-ordered and FE-volume data; 2D Cartesian for IJ-ordered and FE-surface data; XY Line for I-ordered data.
