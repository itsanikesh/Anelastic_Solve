3D Orientation Axis

Specifies whether and how the 3D Orientation Axis is displayed.

 

The following options are available:

  • Show 3D Orientation Axis: Select this check box to display the 3D Orientation Axis.

  • Color: Choose a color for the axis lines from the Select Color dialog.

  • Size (%): Specifies the size of the axis lines in the 3D Frame Axis as a percentage of the current frame width. Either enter a value in the text field, or press either of the increase/decrease buttons to increase or decrease the value. The default is 10.

  • Thickness (%): Specifies the thickness of the axis lines in the 3D Frame Axis as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, 1.5. You can alter these default values from the Size Preferences dialog.

  • Position (%): Specifies the position of the 3D Frame Axis as a percentage of frame width in both the X- and Y-direction within the frame. You can enter new values in the text fields labeled "X" and "Y" respectively. The default is 85, in both the X- and Y-directions.
