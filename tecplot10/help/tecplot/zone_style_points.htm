Zone Style Points

Use the Points page of the Zone Style dialog to select sources for the data points used to plot vectors and scatter symbols. Each zone is listed, along with its name and current attributes.

 

The following options are available:

Zone Num: Select zones by number. The following options are available:

  • Select One: Select a single zone by number. An Enter Value dialog appears after selecting this option. Enter the number of the zone you want to select. To select a single, higher number zone, when you have a large number of zones, use the Select One option. If you have just a few zones, it is usually faster to simply select the zone with the cursor.

  • Select Range: Select a range of zones. An Enter Range dialog appears after selecting this option. Enter the range as the beginning zone, the ending zone, and the skip factor. To select every other map, use a skip factor of two. To select every third map, use a skip factor of three, and so on for other variations. To quickly select a specific range, when you have a large number of zones, use the Select Range option. If you have just a few zones, it is usually faster to simply select the range with the cursor. To select a single zone, click on it.

  • Select All: Select all zones.

·

Zone Name: Select zones by name. An Enter Text String dialog appears. Enter the name of the zone you want to select. Use the asterisk (*) to match any character.

 

Zone Group: Select zones by group, or edit group assignments. By default, all zones are assigned to Group 1. You can create groups of zones which can be modified as one by editing the zone group, then selecting zones by group. The following options are available:

  • Select One: Select a single group. An Enter Value dialog appears. Enter the desired group number. If you select a group number to which no zones are assigned, no zones are selected. Click OK to select all zones in the selected group.

  • Select Range: Select a range of groups. An Enter Range dialog appears after selecting this option. Enter the desired range as a begin group, an end group, and a skip. To select every other group, use a skip factor of two. To select every third group, use a skip factor of three, and so on for other variations. To select all zones in the selected range of groups, click OK.

  • Edit Group Number: Modify the group assignment for selected zones. Select one or more zones, using either the cursor or the Zone Num and Zone Name selection options.

 

Zone Show: Activate or deactivate selected zones. Deactivating a zone turns off the zone completely – no layers are displayed for a deactivated zones. The following options are available under Zone Show:

  • Activate: Show selected zones.

  • Deactivate: Turn off selected zones. There must always be at least one active zone; if you try to deactivate all zones, zone 1 remains (or becomes) active and an error message appears informing you that you cannot deactivate all zones.

  • Show Selected Only: Display individually selected maps together.

  • Invert: Invert the selected map.

 

Points to Plot: Select how points are displayed. Options include:

  • Nodes on Surfaces: Draws only the nodes that are on the surface of the zone.

  • All Nodes: Draws all nodes in the zone.

  • All Connected: Draws all the nodes that are connected by the node map. Nodes without any connectivity are not drawn.

  • Cell Centers Near Surfaces: Draws points at the cell centers which are on or near the surface of the zone.

  • All Cell Centers: Draws points at all cell centers in the zone.

 

Index Skip: Specify the skip intervals for the I-, J-, and K-indices, or specify no skip (the default). For example, if the vector I-Skip is four, only every fourth vector in the I-direction is plotted. If the I-Skip is one, all vectors in the I-direction are plotted. For irregular and finite-element data, only the I-Skip has an effect, skipping through nodes in the order they are listed in the data file. The menu options are as follows:

  • No Skip: Set the I-, J-, and K-skip intervals to one; plot all vectors.

  • Enter Skip: Specify I-, J-, and K-skip intervals on the Enter Index Skipping dialog.
