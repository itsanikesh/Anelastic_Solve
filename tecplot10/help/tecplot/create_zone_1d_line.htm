Create Zone 1D Line

Use the Create 1D Line Zone dialog to create an I-ordered zone consisting of a specified number of points spanning a specified range.

 

The following options are available:

  • Number of Points: Enter the desired number of points for the new zone in this text field.

  • Coordinates: Enter the range of the data in the following text fields:

- XMin: Enter the minimum value of the data range.

- XMax: Enter the maximum value of the data range.

 

To create the zone, click Create.
