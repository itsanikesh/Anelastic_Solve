Edit Current Frame

The Edit Frame dialog gives you precise control over the size and positioning as well as the format of the frame. The size and position options are expressed in terms of paper ruler units; to view the ruler, select the Show Ruler check box from the Workspace Paper dialog.

 

The following options are available:

  • Left Side: Specify the position of the left side of the frame, in paper ruler units.

  • Top Side: Specify the position of the top side of the frame, in paper ruler units.

  • Width: Specify the width of the frame, in paper ruler units.

  • Height: Specify the height of the frame, in paper ruler units.

 

The format options are as follows:

  • Show Border: Select this check box to show the frame border. The frame border appears only if the Show Border check box is selected.

  • Border Thickness: Enter a thickness for the frame border, as a percentage of the frame height.

  • Show Header: Select this check box to show the frame header. The header appears only if both the Show Header and Show Border check boxes are selected.

  • Color: Select a color for the frame header from the Select Color dialog.

  • Show Background: Select this check box to show the frame background. If this box is not selected, the frame is transparent.

  • Color: Select a color for the frame background from the Select Color dialog. The background color appears only if the Show Background check box is selected.

  • Frame Name: Assign a name to the frame. You will be able to push and pop the frame using the name you assign.
