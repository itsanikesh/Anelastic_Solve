Create Mappings

Use the Create Mappings dialog to add a single map or several related maps to the current list of mappings.

 

The following options area available:

  • X-Axis Var versus Y-Axis Var for One Zone: Select this option button to define a single map with one X-axis variable and one Y-axis variable in one zone. You can then supply a Mapping Name, the X-axis variable, the Y-axis variable, and the desired zone.

  • X-Axis Var versus Y-Axis Var for All Zones: Select this option button to define a series of maps, one for each zone, with the specified X-axis and Y-axis variables. If you choose this option, you specify only the X-axis and Y-axis variables; the map names are assigned automatically by Tecplot, and there is one map per zone.

  • X-Axis Var versus All Other Variables: Select this option button to define a series of maps (one fewer than the total number of variables) You specify the X-Axis variable and the desired zone; Tecplot creates maps using each of the remaining variables as Y-axis variables, automatically assigning a map name to each.

  • Y-Axis Var versus All Other Variables: Select this option button to define a series of maps (one fewer than the total number of variables)You specify the Y-Axis variable and the desired zone; Tecplot creates maps using each of the remaining variables as X-axis variables, automatically assigning a map name to each.

  • Mapping Name: If you selected X-Axis Var versus Y-Axis Var for One Zone, you can enter a name for your new map in this text field.

  • X-Axis Var: Unless you have selected Y-Axis Var versus All Other Variables, you can select an X-Axis variable from this drop-down.

  • Y-Axis Var: Unless you have selected X-Axis Var versus All Other Variables, you can select an Y-Axis variable from this drop-down.

  • Zone: Unless you have selected X-Axis Var versus Y-Axis Var for All Zones, you can select the map's zone from this drop-down.
