Quick Edit Field Shade

This area controls whether shading is used for selected zones, and if so, using what plot type.

 

The following options are available:

  • Shade Show: Display light source shading for the selected zones. This option allows you to turn off just the Shade zone layer for specific zones, without completely deactivating the zones. The following options are available:

    • Yes: Show light-source shading for the selected zones. If the Shade zone layer is not currently active, a dialog appears asking if you wish it activated. Click OK to activate the shade layer.

    • No: Turn off light-source shading for the selected zones.
