Sidebar Plot Type

Choose the 3D Cartesian plot type to select a 3-D view of the current data set.

 

Choose the 2D Cartesian plot type to select a 2-D view of the current data set.

 

Choose the XY Line plot type to select an XY view of the current data set.

 

Choose the Polar Line plot type to select a polar view of the current data set.

 

Choose the Sketch plot type to select a Sketch view of the current data set. In the Sketch view, no data is displayed; only text and geometries are displayed.
