Addon Export DXF

The DXF Export add-on will export data in DXF (drawing interchange) format.

 

If the data type is finite-element and the element type is triangular or quadrilateral:

 

  • When working in 3-D, a DXF 3DFACE entity is created for each triangle or quadrilateral.

  • When not working in 3-D, a DXF POLYLINE entity is created for each triangle or quadrilateral.

 

If the element type is not triangular or quadrilateral, then no DXF entities are exported.

 

If the data type is I-, IJ-, or IJK-ordered:

 

If working in 3-D and JMax = 1 and KMax = 1, then a DXF POLYLINE entity is exported for all points in the selected zone(s).

 

If working in 3-D and JMax or KMax is greater than 1, then DXF 3DFACE entities are exported for all points in the selected zone(s).

 

If not working in 3-D, then a DXF POLYLINE entity is exported connecting all points in the selected zone(s).
