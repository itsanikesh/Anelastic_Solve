Keyboard Shortcuts

Extended Mouse Operations

The middle and right mouse buttons are powerful tools you may use to immediately zoom and translate your data without having to switch to the Zoom or Translate tools on the sidebar. This advanced mouse/keyboard functionality is available when using any 3D rotate, Contour, Geometry (except Polyline), Probe, Slice, Streamtrace Placement, Translate, Zoom, or Zone Creation tools. If you have a two-button mouse use the Ctrl key in conjunction with the right mouse button to achieve middle mouse button capabilities.

 

Middle Mouse Button/Ctrl-Right Mouse Button

  • Click: Redraw. If the pointer is in the current frame then the frame is redrawn, otherwise, redraw all frames. (This is the default action for a click. It may be configured with the $!INTERFACE MOUSEMIDDLEBUTTONMODE command.)

  • Drag: Smoothly zoom in or out. An upward motion zooms out. An upward motion zooms in.

  • Alt-Drag: In the 3D Cartesian plot type, move the viewer further from (upward motion) or closer to (downward motion) the object. In all other plot types, this behaves like the Drag action.

 

Right Mouse Button

  • Click: Switch from the current tool to the Selector tool. (This is the default action for a click. It may be configured with the $!INTERFACE MOUSERIGHTBUTTONMODE command.)

  • Drag: Translate.

  • Alt-Drag: Same as the Drag action.

 

Mouse Tool Operations

The following contains all mouse/keyboard operations you may use with various sidebar tools. (All mouse button operations utilize the left button.)

 

3D Rotate Tools

  • Drag: Rotate about the defined rotation origin with your current Rotate tool.

  • Alt-Drag: Rotate about the viewer position using your current Rotate tool.

  • C: Move rotation origin to probed point, ignoring zones.

  • O: Move rotation origin to probed point of data.

  • R: Rollerball rotation.

  • S: Spherical rotation.

  • T: Twist rotation.

  • X: X-axis rotation.

  • Y: Y-axis rotation.

  • Z: Z-axis rotation.

 

Contour Add Tool

  • Alt-Click: Place a contour line by probing on a streamtrace, slice, or iso-surface.

  • Click: Place a contour line.

  • Ctrl-Click: Replace the nearest contour line with a new line.

  • Drag: Move the new contour line.

  • -: Switch to the Contour Remove tool.

 

Contour Remove Tool

  • Click: Removes the contour line nearest to the probed location.

  • -: Switch to the Contour Add tool.

 

Geometry Polyline Tool

  • A: Allow translation or polyline segments in all directions.

  • H: Restrict translation or current polyline segment to horizontal.

  • U: Pen up, while drawing polyline.

  • V: Restrict translation or current polyline segment to vertical.

 

Probe Tools

  • Click: If the pointer is over a valid cell, return the interpolated field values from all nodes in the cell. If multiple cells are candidates: in 2-D the cell from the highest number zone is used; in 3-D the cell closest to the viewer is used.

  • Ctrl-Click: If the pointer is over a valid cell, return the field values from the nearest node in the cell. If multiple cells are candidates: in 2-D the cell from the highest number zone is used; in 3-D the cell closest to the viewer is used. If the pointer is not over any cell, the field values from the nearest data point as measured in distance on the screen are used.

  • Shift-Ctrl-Click: Return the field values from the nearest point on the screen, ignoring surfaces, zone number, or depth of the point. (This is useful in 3-D probing on data points that are on the back side of a closed surface without having to rotate the object. In 2-D this is useful for probing on data points for zones that may be underneath other zones because of the order in which they were drawn.)

  • Alt-Click: Same as Click except ignore zones while probing. (Probe only streamtraces, iso-surfaces, and slices.)

  • Alt-Ctrl-Click: Same as Ctrl-Click except ignore zones while probing. (Probe only streamtraces, iso-surfaces, and slices.)

  • Alt-Ctrl-Shift-Click: Same as Shift-Ctrl-Click except ignore zones while probing. (Probe only streamtraces, iso-surfaces, and slices.)

 

Slice Tools

  • +: Turn on the start slice if no slices are active, or turn on the end slice if slices are already active.

  • -: Turn off the end slice if the end slice is active, or turn off the start slice if the end slice is not active.

  • Click: Place a start slice.

  • Drag: Move the start slice.

  • Alt-Click/Alt-Drag: Determine the XYZ-location by ignoring zones and looking only at derived volume objects (streamtraces, iso-surfaces, and slices.)

  • Shift-Click: Place the end slice.

  • Shift-Drag: Move the end slice.

  • I, J, K (ordered zones only): Switch to slicing constant I-, J-, or K-planes respectively.

  • X, Y, Z: Switch to slicing constant X-, Y-, or Z-planes respectively.

  • 0-9: Numbers one through nine activate intermediate slices and set the number of intermediate slices to the number entered; zero turns off intermediate slices.

 

Streamtrace Placement Tools (3-D Only)

  • D: Switch to streamrods.

  • R: Switch to streamribbons.

  • S: Switch to surface lines.

  • V: Switch to volume lines.

  • 1-9: Change the number of streamtraces to be added when placing a rake of streamtraces.

 

Translate/Magnify Tool

  • Drag: Translate the data.

  • Shift-Drag: Translate the paper.

  • -: If the drag was started with Shift, this will reduce the magnification of the paper, otherwise, this will reduce the magnification of the data.

  • +: If the drag was started with Shift, this will increase the magnification of the paper, otherwise, this will increase the magnification of the data.

  • - Drag: Decrease magnification of the paper.

  • + Drag: Increase magnification of the paper.

 

Zoom Tool

  • Click: Center the zoom around the location of your click.

  • Picked Object Options

  • +: Reduce the size of the object. If multiple objects are selected, all object positions will be shifted towards the first object selected.

  • -: Increase the size of the object. If multiple objects are selected, all object positions will be shifted away from the first object selected.

  • Del: Delete picked object(s).

  • Ctrl-C: Copy picked object(s) to the clipboard.

  • Ctrl-V: Paste picked object(s) from the clipboard.

  • Ctrl-X: Cut picked object(s).

 

Other Keyboard Operations

  • Ctrl-A is Paste View: Copy the contents of the view buffer into the current frame.

  • Ctrl-C is Copy: Store the selected item in the Paste buffer. The Paste buffer is specific to Tecplot; you cannot cut objects from Tecplot and paste them into other applications.

  • Ctrl-D is Redraw All: Completely regenerates the Tecplot workspace, including all the frames. The Redraw All button is the sidebar shortcut.

  • Ctrl-F is Fit to Full Size: Fit the entire plot into the frame, including all data, text, and geometries.

  • Ctrl-L is Last: Restore previous view from the Tecplot view stack.

  • Ctrl-O is Open Layout: Calls up the Open Layout dialog, used to restore a previously saved Tecplot layout. Also used to open a layout with a different set of data.

  • Ctrl-P is Print: Calls up the Print dialog, which controls Tecplot printing.

  • Ctrl-Q is Exit: Quit Tecplot and return to your operating system.

  • Ctrl-R is Redraw: Redraw the current frame to show all pending changes.

  • Ctrl-S is Save Layout: Saves the current layout as a layout file, or if the current layout was previously saved, it updates the layout file.

  • Ctrl-U is Publish: Calls up the Publish dialog, which controls Web publishing.

  • Ctrl-V is Paste: Add the contents of the Paste buffer to the current plot. Pasting an object into the same frame that it was cut or copied from copies the object to the location it came from, that is, the new copy is placed directly on top of the original object. Use the Selector or the Adjustor to move the copy to different locations in the frame.

  • Ctrl-W is Save Layout As: Save a previously saved layout under a new name.

  • Ctrl-X is Cut: Remove the selected item from the plot, and the current data set (if applicable), but store the removed item in the Paste buffer. The Paste buffer is specific to Tecplot; you cannot cut objects from Tecplot and paste them into other applications.
