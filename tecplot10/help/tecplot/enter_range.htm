Enter Range

Use the Enter Range dialog to specify a range of values. This dialog has the following fields:

  • Maximum (Display only): Shows the maximum allowable value for the range.

  • Begin: Enter the beginning value for the range.

  • End: Enter the end value for the range.

  • Skip: Enter a Skip factor (one means use all values in the range, two means use every other value, three means use every third value, and so on).
