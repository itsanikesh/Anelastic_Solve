Probe Specific Values

The Probe dialog for XY and Polar Line plots has two different forms, depending on whether you are probing for the interpolated value of the dependent variable or for information on the nearest data point.

 

For interpolated plot values, the Probe dialog lists every mapping currently active for the current data set, together with the value of the dependent variable at the specified probe point. If you are probing along a Y-axis and no active maps have X as a dependent variable, all the value columns read "...." By default, each map is shown on a single line, which allows display of about the first ten characters of the map name and seven significant digits of the variable value.

 

Below the mappings the position of the probe is shown.

  • Theta/X-Value: Theta- or X-value of the nearest data point to the probe position.

  • R/Y-Value: R- or Y-value of the nearest data point to the probe position.

  • I-Index: I-index of the nearest data point to the probe position.

  • J-Index: J-index of the nearest data point to the probe position.

  • K-Index: K-index of the nearest data point to the probe position.

  • Map: Number and name of the nearest map to the probe position.

  • Zone: Number and name of the nearest zone to the probe position.

  • IMax: Maximum I-index of the current zone.

  • JMax: Maximum J-index of the current zone.

  • KMax: Maximum K-index of the current zone.

  • X-Axis: X-axis associated with the current map.

  • Y-Axis: Y-axis associated with the current map.

 

To call up the Probe At dialog select the Probe At option from the Data menu, or click Probe At on the sidebar.
