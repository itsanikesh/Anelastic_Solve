Mapping Style Definitions

Use the Definitions page of the Mapping Style dialog to set up mappings, which associate a pair of variables with a pair of axes.

 

To select a range of maps:

  1. Click on the first (or last) map.

  2. Shift-click on the last (or first) map in the range.

 

To select an arbitrary collection of maps:

  1. Click on the first map.

  2. Ctrl-click on subsequent maps until all desired maps are selected.

 

When the select variable dialog appears after selecting this option:

  1. Choose one of the data set variables from the drop-down.

  2. Click OK.

 

When the select variable dialog appears after selecting this option:

  1. Choose one of the data set variables from the drop-down.

  2. Click OK.

 

When the select zone dialog appears after selecting this option:

  1. Choose one of the data set zones from the drop-down.

  2. Click OK.

 

For more information, please refer to the Tecplot Users Manual.
