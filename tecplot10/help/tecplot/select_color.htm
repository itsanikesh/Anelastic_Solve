Select Color

Use the Select Color dialog to choose a color.

 

You may be able to select Multi when working with meshes, contours, vectors, scatters, or other objects. Multi C1, C2, C3 or C4 are used to color an object using the contour variables C1, C2, C3 and C4 respectively. For example, using Multi C1 for a mesh plot creates a multi-colored mesh plot which varies in color with the C1 contour variable. RGB is used to color an object using the Red, Green, Blue variables. The color of each segment is determined from the average value of the variable(s) at its two end points, and various other settings such as the number of levels.

 

You may edit a color with the Color Preferences dialog, which may be selected by choosing Colors on the Preferences sub-menu of the File menu.
