Preferences Miscellaneous

Use the Preferences menu to select options for setting the control size, color, and other Tecplot preferences.

  • Allow Old Text Formatting: Selecting this option causes Tecplot to process text formatting symbols in layout files created with previous versions of Tecplot. For example, a character proceeded by a caret (^) will be drawn superscripted. Tecplot 10 replaces this character-by-character formatting with HTML-like tags enclosing the formatted characters. Refer to help on the Text Details dialog for more information.

  • Allow Data Sharing: Selecting this option enables zones to share variables and connectivity. If a variable or connectivity list is shared, then only a single copy of it exists and is used by two or more zones. Refer to help on the Sharing page of the Data Set Info dialog for more information on data sharing.
