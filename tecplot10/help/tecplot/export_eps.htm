Export EPS

Encapsulated PostScript (EPS) is a special type of PostScript file designed for inclusion in other applications.

 

The following options are available:

  • Color: Select this check box to save the image as a color EPS file.

  • Render Type:

  • Vector: Select this option to export your EPS in a vector format.

  • Image: Select this option to export your EPS in an image format.

  • Region: Select Current Frame, All Frames, or Work Area from the drop-down. Current frame is the default and captures only the current frame. All Frames captures the smallest rectangular area containing all frames. Work Area captures the workspace.

  • Resolution: Set resolution in terms of dots per inch for image output.

  • Extra Precision: Specify the number of decimal places to which size and position parameters are carried. The number of decimal places added to the integer value will be used as coordinates for the resulting vector-based print output. Use the increase or decrease arrows, or enter a value in the text field. (Range 1-8.)

  • Force Extra Sorting for 3D Frames: Selecting this check box will cause Tecplot to use extra sorting in all 3-D frames for vector output.

 

EPS Preview Image Settings

  • Type: Select the type of preview image. Chose from None, TIFF, EPSIV2, or FrameMaker. Selecting None means no embedded image information is added to the EPS file. If no preview image is available, importing applications display the imported file as a box outline, a gray rectangle, or blank space. Selecting TIFF will include a monochrome/gray scale TIFF image in the EPS file. This is the most common image format. You can specify bit depth for the TIFF image by selecting one of the options under Depth. Selecting EPSIV2 will include a monochrome (one bit per pixel) EPS Version 2 image. Selecting FrameMaker will include a monochrome (one bit per pixel) image appropriate for importing into older versions of FrameMaker.

  • Depth: Available only when TIFF is selected in they Type drop-down. Chose from On/Off, 1-Bit Gray, 4-Bit Gray, or 8-Bit Gray. On/Off uses one bit per pixel, with the pixel off for black, and on for any other color. One-Bit Gray uses one bit per pixel, with the pixel off if the gray scale equivalent of the color is less than 50 percent white, on if more than 50 percent. Four-Bit Gray uses four bits per pixel to allow moderate gray scale. Eight-Bit Gray uses eight bits per pixel to allow full gray scale.

  • Width (pixels): Enter a value in the text field for your exported image’s width. The image region is rendered to the image file to exactly fit a size of Width by Height. (Default is 128.)

  • Height (pixels): Enter a value in the text field for your preview image’s height. (Default is 128.)
