Select All

Use the Select All dialog to select all of a specific type of Tecplot objects.

 

Choosing Select All displays a dialog with a list of object types for you to select with this command.

 

The available object types are:

  • Frames: Select all frames in the current plot.

  • Zones: Select all zone data in the current plot.

  • Text: Select all text in the current plot.

  • Geometries: Select all geometries (circles, ellipses, squares, rectangles, and polylines) in the current plot.

  • Streamtraces: Select all streamtraces in the current plot.

 

Click on the all the check boxes for each of the specific object types you want and select the current plot. Then, click OK to select all items of the chosen types, or Cancel to dismiss the dialog without selecting any items.
