Extract Streamtraces

Use the Extract Streamtraces dialog to extract streamtraces into zones. Any type of streamtrace can be extracted to form zones. Use of this option is discussed in the topic Extracting Streamtraces into Zones.

 

The following option is available:

  • Concatenate Common Streamtraces into One Zone: Select this check box to concatenate all streamtraces of a given type into a single zone. For example, if you placed three points and a rake of a volume line, then two points and four rakes of volume rod, and then a rake of volume line, you would get three zones (one for the first set of volume line, one for the set of volume rod, and one of the second set of volume line). If you choose not to concatenate streamtraces, you get one zone per streamtrace.

 

The zones created are I- or IJ-ordered zones if not concatenated. If streamtraces are concatenated, FE line segment or FE quadrilateral zones are created. (Created on the Timing page of the Streamtrace Details dialog.)

 

After extracting streamtraces into zones, you may plot them like any other zone.

 

Finite-element zones are also created when streamtraces are dashes.
