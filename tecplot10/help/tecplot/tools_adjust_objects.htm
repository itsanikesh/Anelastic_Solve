Tools Adjust Objects

Use the Adjustor tool to perform the following specific modifications to your plot and data:

  • Change the location of individual or groups of data points in the grid.

  • Modify the values of the data set variables at a particular point.

  • Change the length or placement of individual axes (2D Cartesian and XY Line plot types only).

  • Change the spacing between an axis label and its associated axis (2D Cartesian and XY Line plot types only).

  • Change the shape of a polyline.

  • Except for these specific actions, the behavior of the Adjustor mode is identical to that of the Selector tool.

 

Note: The Adjustor tool can alter your data. Be sure you want to use the Adjustor tool before dragging points in the data region.

 

To change the length of an axis:

  1. Select the Adjustor tool.

  2. Select on the end point handle with the Adjustor.

  3. Drag the end point to the desired position. The cursor changes to a resize cursor.

  4. Drag the handle until the axis is the desired length.

 

To change the placement of the axis:

  1. Select the axis with the Adjustor.

  2. Move the cursor over the axis until the cursor becomes a bi-directional arrow.

  3. Drag the axis to the desired location.

 

To change the spacing between an axis label and its associated axis:

  1. Select the Adjustor tool.

  2. Select the axis label with the Adjustor. The cursor changes to a bi-directional arrow.

  3. Drag the label to the desired location.

  4. Modify the values of the data set variables at a particular point.

 

To modify the values of the data set variables at a particular point:

  1. Select the Adjustor tool.

  2. Double-click on the desired point or single-click and then select Object Details from the sidebar. The Probe dialog appears; use it to view or modify the values of the data set variables.

 

Select this option to permit the actual moving of data points with the Adjustor tool. By default, this option is not selected. When selected, the Adjustor cannot move data points in a plot. This option does not affect any of the Adjustor’s other capabilities.
