Import Excel

The Excel Data Loader

 

Excel Data Loader can read numeric and text data from Microsoft Excel Version 5.0 or higher, .xls files and import the data into Tecplot. Excel Data Loader is available only for Windows platforms.

 

Excel Data Loader can process information in the spreadsheet in four ways. You can specify table format, carpet format, or a user-defined format in which you specify the zone data locations and properties. The definitions of these formats are:

  • Table: Data is arranged in columns, with a variable name at the head of each column. Be certain that the loader has accurately specified the block of data that you want to load.

  • Carpet: Data is arranged with X-grid coordinates in the first row, Y-grid coordinates in the first column, and values of a variable V, measured at or calculated for those (X, Y) locations, in the interior of the spreadsheet.

  • Other (or if the loader did not identify any carpet or table blocks): A prompt appears to choose Default or Custom data format.

  • The Default format option assumes a single zone of I-ordered, POINT format data.

  • The Custom option allows you to load you spreadsheet by zone, as ordered or finite element data, and as a partial load.

 

Once you have selected Other, Custom and Next, enter:

  • The cell numbers of data you wish to load.

  • The format of the data.

  • For finite-element data, the number of nodes, number of elements, and geometry of elements.

  • Zones and their cell numbers.
