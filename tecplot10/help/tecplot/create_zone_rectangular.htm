Create Zone Rectangular

Use the Create Rectangular Zone dialog to create an IJ-ordered or IJK-ordered zone with specified dimensions and variable ranges. To create an IJ-ordered zone, call up this dialog in the 2D Cartesian or Sketch plot types. To create an IJK-ordered zone, call up this dialog in the 3D Cartesian or Sketch plot types.

 

The following options are available

  • Dimensions: Enter the desired number of points in each dimension in the following text fields (the total number of points is the product of all the specified dimensions):

o I: Enter the number of points in the I-dimension.

o J: Enter the number of points in the J-dimension.

o K: Enter the number of points in the K-dimension. (Available in the 3D Cartesian plot type only.)

  • Coordinates: Enter the range of the data in the following text fields:

o XMin: Enter the minimum X-value of the data range.

o XMax: Enter the maximum X-value of the data range

o YMin: Enter the minimum Y-value of the data range.

o YMax: Enter the maximum Y-value of the data range.

o ZMin: Enter the minimum Z-value of the data range. (Available in the 3D Cartesian plot type only.)

o ZMax: Enter the maximum Z-value of the data range. (Available in the 3D Cartesian plot type only.)
