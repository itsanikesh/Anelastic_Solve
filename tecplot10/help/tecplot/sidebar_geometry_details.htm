Sidebar Geometry Details

Use the Geometry Details dialog to specify attributes of polylines, circles, ellipses, squares and rectangles.

 

To access the Geometry dialog either select a geometry and click on Object Details in the Tecplot sidebar, or double-click on the geometry object itself.

 

The following options are available:

  • Line Color: Select a color for the geometry from the Select Color dialog.

  • Line Pattern: Select the desired pattern from among the following: Solid, Dashed, DashDot, Dotted, LongDash, DashDotDot.

  • Pattern Length (%): Specifies the length of the line pattern as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.5, 0.8, 2, 3 and 5. You can alter these default values from the Size Preferences dialog.

  • Line Thickness (%): Specifies the thickness of the line as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1. 0.4, 0.8 and 1.5. You can alter these default values from the Size Preferences dialog.

  • Fill Color: Select this check box to fill a circle, ellipse, square, rectangle or line segment polygon. Then select a color for the geometry fill from the Select Color dialog.

  • Origin: Enter the X- and Y-coordinates of the anchor position of the geometry. You use frame units (percentage of frame width for X, percentage of frame height for Y) if the coordinate system is Frame; you use grid units if the coordinate system is Grid.

  • Coordinate System: Specify the coordinate system for the geometry, either Frame or Grid. If you choose Frame, the geometry is always displayed at constant size when you zoom in or out of the plot. If you choose Grid, the geometry resizes with the data grid.

  • Attach to Zone/Map: Select this check box to attach the geometry to a particular zone or mapping. Geometries that are attached to an inactive or non-existent zone are not displayed. If you select this check box, enter the number of the zone or mapping to which you want to attach the geometry.

  • Show in All "Like" Frames: Select this check box to display the geometry in all frames sharing the current frame's data set.

  • Macro Function: In the text field, specify the name of the macro function that you wish to link to a particular geometry

 

The following fields specific to a single geometry type appear when appropriate:

  • Polyline Arrowhead: These options control the appearance of an arrowhead on a drawn polyline.

  • Attachment: Choose the end or ends of the polyline by selecting the appropriate check boxes. Choose either or both of Start and End.

  • Size(%): Specify the size of the arrowhead, as a percentage of frame height. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 1, 3, 5, 8 and 12. You can alter these default values from the Size Preferences dialog.

  • Style: Choose from one of the following: Plain (lines drawn from point of attachment), Filled (a filled triangle), Hollow (a hollow triangle).

  • Angle: Specify the angle the arrowhead makes with the polyline. You can either enter a value (in degrees) in the text field, or choose a preset value from the drop-down.

  • Circle: Controls the radius and precision of approximation of the circle, as follows:

  • Radius: Change the radius of the circle. Enter the radius of the circle, in coordinate system units, either Frame or Grid.

  • Approximated by Number of Sides: Enter the number of polylines used to approximate the circle.

  • Ellipse: Controls the shape and precision of approximation of the ellipse, as follows:

  • Horizontal Axis: Change the horizontal axis of the ellipse. Enter the length of the horizontal axis in coordinate system units, either Frame or Grid.

  • Vertical Axis: Change the vertical axis of the ellipse. Enter the length of the vertical axis in coordinate system units, either Frame or Grid.

  • Approximated by Number of Sides: Enter the number of polylines used to approximate the ellipse.

  • Square: Controls the size of the square, as follows:

  • Size: Change the size of the square. Enter the width of the square, in coordinate system units, either Frame or Grid.

  • Rectangle: Controls the size and shape of the rectangle as follows:

  • Width: Change the width of the rectangle. Enter the width of the rectangle in coordinate system units, either Frame or Grid.

  • Height: Change the height of the rectangle. Enter the height of the rectangle in coordinate system units, either Frame or Grid.
