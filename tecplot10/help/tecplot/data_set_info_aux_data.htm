Data Set Info Aux Data

The Data Set Information dialog, accessed from the Data Set Info option on the Data menu, gives summary information about the current data set.

 

The Aux Data page of the Data Set Information dialog offers the following options:

  • Show Auxiliary Data: Use the drop-down to display auxiliary data for data sets, zones, vars, line maps, frames or by name. If Zone or Name is selected, a second drop-down is available for further selection (zone or auxiliary variable name).

  • Data Name/Value: Displays the names and values of any auxiliary data.
