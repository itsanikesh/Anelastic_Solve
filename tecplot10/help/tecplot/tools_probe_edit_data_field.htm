Tools Probe Edit Data Field

Use the Probe Edit Data dialog to modify the variable values for a single data point.

 

The following options are available:

  • Variable: Lists all variables in the current data set. If there are more than ten variables, the Scroll Up and Scroll Down buttons are active.

  • Value: Lists the value of the named variable at the probed point.

  • Scroll Up: Click this to scroll up one page of variables. This button is active only if there is more than one page of variables.

  • Scroll Down: Click this to scroll down one page variables. This button is active only if there is more than one page of variables.

  • Enter Zone/Index to Probe: Specify a grid point by index for editing. There are four controls in this region:

  • Zone: Select a zone to probe from the drop-down.

  • I: Specify the I-index of the probed point. You can either enter a value, or use the up and down arrows to increase or decrease the current value.

  • J: Specify the J-index of the probed point. You can either enter a value, or use the up and down arrows to increase or decrease the current value.

  • K: Specify the K-index of the probed point. You can either enter a value, or use the up and down arrows to increase or decrease the current value.

 

To change the value of a variable at the selected point:

  1. Enter a number in one of the value text boxes.

  2. Click Close.
