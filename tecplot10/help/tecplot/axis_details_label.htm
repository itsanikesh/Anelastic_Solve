Axis Details Label

Use the Labels page of the Axis Details dialog to control the drawing of tick mark labels.

 

The following options are available:

  • Show Labels On: Select the appropriate check boxes for label display. Options depend upon the plot type setting and include Axis Line, Inner Circle, Outer Circle, All R-Axes, Grid Border Left, Grid Border Right, Grid Border Top, Opposite Edge, Grid Border Start, Grid Border End.

  • Color: Choose the color for the tick mark labels from the drop-down of Tecplot's basic colors.

  • Font: Choose the font and size for the tick mark labels using the Select Font dialog, or use the up/down arrows to increase/decrease the font size.

  • Number Format: Specify the print format for the tick mark labels on the Specify Number Format dialog which appears.

  • Offset from Line (%): Enter the offset of the tick mark labels from the axis.

  • Orient Labels: Specifies additional options for arranging tick mark labels orientation.

    • At Angle: Labels are set at an angle you specify.

    • Parallel to Axis: Labels are set parallel to the axis.

    • Perpendicular to Axis: Labels are at right angles (perpendicular) to the axis.

  • Angle (deg): Specify the orientation of the tick mark labels relative to the axis. The angle is measured in degrees counter-clockwise from horizontal. Horizontal numbers are at zero degrees; vertical numbers are at 90 degrees.

  • Show Label at Axis Intersection (Not available for 3D Cartesian plot types): When not checked, Tecplot does not show labels that would intersect any R-axis line or the R-axis border (if these are on).

  • Erase Behind Labels (Not available for 3D Cartesian plot types): When checked, the area behind each label is erased with the frame background color.

  • Label Skip: Specifies the interval at which ticks are to be labeled. For instance, two marks every other major tick mark.

  • Tick Mark and Label Spacing: Controls the spacing of the tick marks and tick mark labels. Options include:

    • Auto Spacing: Select this check box to turn on automatic tick mark and tick mark label spacing. Tecplot automatically determines the number of tick mark labels and their spacing.

    • Manual Spacing: If Auto Spacing is not selected, two manual spacing options are activated. These options are:

      • Spacing: Enter the distance between tick marks (in grid units) in the field.

      • Anchor: Specify an anchor value in the field. The anchor value specifies where one tick mark will be placed, and other tick marks are placed relative to this point.

 

Both the Spacing and Anchor fields are ignored if Auto Spacing is selected.
