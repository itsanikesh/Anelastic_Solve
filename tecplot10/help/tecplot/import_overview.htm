Import Overview

Tecplot allows you to load data in a number of formats. These include:

  • Excel.

  • PLOT3D.

  • Text Spreadsheet Loader.

  • Gridgen.

  • DXF.

  • HDF.

  • HDF-5.

  • DEM.

  • Fluent.

  • PLY Polygon.

  • CGNS.

  • General Text.

 

Visit our Web site at www.tecplot.com for any additional loaders built after the Version 10.0 release.

Once you have loaded your data, it will generally be up to you to use Tecplot's features to produce a plot that suits your purposes.
