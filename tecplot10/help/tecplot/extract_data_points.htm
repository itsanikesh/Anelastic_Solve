Extract Data Points

Use the Extract Data Points dialog to specify how to extract data points from a selected polyline, or to specify how to dispose of points extracted using the Extract Points or Extract Polyline modes.

 

The following options are available:

  • Extract Data To:

    • File: Select this if you would like the data points extracted to a ASCII Tecplot data file. If this check box is selected, the Extract Data Points to File dialog appears when you click Extract. Use this dialog to specify a file name for the extracted data file.

    • Zone: Select this if you would like the data points extracted to a zone.

  • Include Distance Variable: Select this check box if you want the extracted data file to contain an additional variable, Distance, which contains the accumulated distance from the first point.

  • Number of Points to Extract: Enter the number of points to extract. This field is not editable if you have selected Extract Discrete Points from the Data Menu or selected Extract Only Points which Define Geometry on this dialog.

  • Extract Regular Points Along Geometry: Select this check box if you want to extract the specified number of points distributed uniformly along the polyline. This check box is accessible if you are extracting points from a selected polyline using Extract Points from Geometry option or the Extract from Polyline option.

  • Extract Only Points which Define Geometry: Select this check box if you want to extract only the end points of the segments in the geometry.

 

After specifying any desired options, click Extract.
