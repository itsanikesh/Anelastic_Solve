Copy Style Options

Use the Copy Style Options dialog to specify which portions of the current frame style to save.

 

Select one of the following check boxes to store the associated portion of the plot in a stylesheet:

  • Plot Style: Store current plot style in the stylesheet.

  • Text: Store current text in the stylesheet.

  • Geometries: Store current geometries in the stylesheet.

  • Stream Positions: Store current streamtrace positions in the stylesheet.

  • Contour Levels: Store current contour levels in the stylesheet.

  • Include Factory Defaults: Store default settings in the stylesheet.
