Line Legend

Specifies whether, and how, legends are displayed for line plots.

 

The Line Legend options fall into four categories:

  • Show: Specify whether the legend itself or the mapping names are shown.

  • Text: Specify the color, font, and size of text in the legend.

  • Position: Specify the frame position and line spacing of the legend.

  • Legend Box: Specify the attributes of the legend box.

 

The Show options are as follows:

  • Show Legend: Select this check box to place a legend on the line plot.

  • Show Mapping Names: Select this check box to have the legend show the mapping names.

 

The Text options are as follows:

  • Color: Select the text color from the Select Color dialog.

  • Font: Select the text font from the drop-down of Tecplot fonts.

  • Size (%): Specify the text size as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 1.5, 2, 3, 6, and 10. You can alter these default values from the Size Preferences dialog.

 

The Position options are as follows:

  • X (%): Enter the X-position of the anchor point as a percentage of the frame width.

  • Y (%): Enter the Y-position of the anchor point as a percentage of the frame height.

  • Anchor: Launches the Anchor Alignment dialog, where you can specify the anchor, or fixed point of the legend.

  • Line Spacing: Enter the spacing between lines in the legend, in lines. The line spacing times the Size (%) field is the approximate distance between lines in frame units.

 

The Legend Box options are as follows:

  • Legend Box: Specifies whether and how the legend box is displayed, as follows:

    • No Box: Select this option to specify that no box is drawn around the legend.

    • Filled: Select this option to specify a filled box around the legend.

    • Plain: Select this option to specify a plain box around the legend.

  • Line Thickness (%): Specifies the thickness of the legend box as a percentage of the frame width. You can enter a value in the text field, or choose one of the values in the drop-down. The default values available are: 0.02, 0.1, 0.4, 0.8, 1.5. You can alter these default values from the Size Preferences dialog.

  • Box Color: Select the box outline color from the Select Color dialog.

  • Fill Color: Select the box fill color from the Select Color dialog.

  • Margin: Specify the margin as a percentage of the legend box width.
