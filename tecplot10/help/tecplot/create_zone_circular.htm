Create Zone Circular

Use the Create Circular Zone dialog to create an IJ-ordered or IJK-ordered zone with specified dimensions and circular coordinates. To create an IJ-ordered zone, call up this dialog in the 2D Cartesian plot type. To create an IJK-ordered zone, call up this dialog in the 3D Cartesian plot type.

 

The following options are available:

  • Dimensions: Enter the desired number of points in each dimension in the following text fields (the total number of points is the product of all the specified dimensions):

    • I (Radial): Enter the number of points in the I-dimension. For a circular zone, I specifies the number of points in the radial direction, that is, the number of circles.

    • J (Circumferential): Enter the number of points in the J-dimension. For a circular zone, J specifies the number of points in the circumferential direction, that is, the number of points on each circle.

    • K (Top to Bottom): Enter the number of points in the K-dimension. (Available in the 3D Cartesian plot type only.)

  • Coordinates: Enter the range of the data in the following text fields:

  • Radius: Enter the radius of the created zone.

  • X-Origin: Enter the X-coordinate of the origin of the created zone.

  • Y-Origin: Enter the Y-coordinate of the origin of the created zone.

  • ZMin: Enter the minimum Z-value of the data range. (Available in the 3D Cartesian plot type only.)

  • ZMax: Enter the maximum Z-value of the data range. (Available in the 3D Cartesian plot type only.)
