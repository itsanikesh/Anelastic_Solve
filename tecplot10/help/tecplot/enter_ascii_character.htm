Enter ASCII Character

Use the Enter ASCII Character dialog to specify an ASCII character to use as a plotting symbol in either a field scatter plot or an X-Y symbol plot. You enter the desired character, and then choose the character set from which to draw the character.

 

This dialog has one text field for specifying the ASCII character and four option buttons representing the available character sets, as follows:

  • Enter Character to use as a Symbol: Enter the desired ASCII character in this text field.

  • Base: Select this to use the English-text character set as the source of the plotting character.

  • Math: Select this to use the math character set as the source of the plotting character.

  • Greek: Select this to use the Greek character set as the source of the plotting character.

  • User-Defined: Select this to use the user-defined character set as the source of the plotting character.
