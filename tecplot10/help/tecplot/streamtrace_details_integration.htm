Streamtrace Details Integration

Use the Integration page of the Streamtrace Details dialog to control the step size in Tecplot's predictor-corrector integration algorithm for calculating streamtraces.

 

The idea of the algorithm is to create the streamtrace by moving in a series of small steps in the direction of the local vector field. Each step is a specified fraction of a cell or element. Tecplot automatically adjusts the step size of the integration based upon the local cell shape and vector field variation.

 

For more information, please refer to the Tecplot Users Manual.
