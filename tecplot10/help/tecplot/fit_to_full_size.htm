Fit to Full Size

Ctrl-F is the keyboard shortcut for Fit to Full Size.

 

This menu changes the view so that all of the data in the active zones and any grid-mode geometries and text are included in the frame.

 

The results of applying Fit to Full Size to 2- and 3-D axes plots are:

  • 2D Independent Axis Mode: Allows the selected region to expand to exactly fit in the frame. The axes are rescaled independently to fit the zoom box.

  • 2D Dependent Axis Mode: Results in the plot being resized to fit the longest dimension of the zoom box to the frame (since the relative axis scale is fixed).

  • 3D Axis Mode: Only changes the size (or magnification) of the data and axes projected onto the frame. The plot is resized such that the longest dimension of the zoom box fits into the frame.
