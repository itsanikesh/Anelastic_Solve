Write XY Data as Formatted Text

 

Data from an XY Line Plot will be written to a text file with the format *.csv for comma separated data or *.txt for any other data separators.

 

All frames with an XY Line Plot style are listed under Choose XY Frames.   You can choose whether frame names, variables names and additional text from the data will be included in the text file.  Values will be written in the “best format”, except variables named Time or Date. 

 

The following options for choosing frames to write are allowed:

 

Current Frame

·        You can choose whether to include data from all data sets (zones), or select data sets from the list.

·        You can choose whether to include all variables, or select variables  from the list.

·        If you toggle off All for data sets or variables, but you do not select anything from the list, all will still be written. 

·        If you select this option when the current frame doesn’t have the XY Line Plot type, you will get an error message.  

 

Selected Frames

Since different frames may have different data sets and different variables, the user is not allowed to select data sets or variables.  All data sets and all variables will be written for each frame selected. 

 

All Frames

Since different frames may have different data sets and different variables, the user is not allowed to select data sets or variables.  All data sets and all variables will be written for each frame.
